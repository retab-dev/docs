---
title: 'Delete Classification'
openapi: 'DELETE /v1/classifications/{classification_id}'
---

Delete a persisted `Classification` record by ID. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()
client.classifications.delete("cls_01G34H8J2K")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();
await client.classifications.delete("cls_01G34H8J2K");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/classifications/cls_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```text 204
No Content
```

```json 404
{
  "detail": "Classification not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="classification_id" type="string" required>
  The unique identifier of the classification to delete.
</ParamField>
