---
title: "Delete Classification"
openapi: "DELETE /v1/classifications/{classification_id}"
---

Delete a classification by ID. The deleted classification no longer appears in list results or retrieval calls.

<RequestExample>

```python Python
from retab import Retab

client = Retab()
client.classifications.delete("cls_01G34H8J2K")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });
await client.classifications.delete("cls_01G34H8J2K");
```

	}

	}
}
