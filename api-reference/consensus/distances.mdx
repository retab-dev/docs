<RequestExample>

```python python
from retab import Retab

client = Retab()

# Two dictionaries to compare
dict1 = {
    "title": "Quantum Algorithms in Interstellar Navigation",
    "authors": ["Dr. Stella Voyager", "Dr. Nova Star", "Dr. Lyra Hunter"],
    "year": 2025,
    "keywords": ["quantum computing", "space navigation", "algorithms"]
}

dict2 = {
    "title": "Quantum Algorithms for Interstellar Navigation",
    "authors": ["Dr. S. Voyager", "Dr. N. Star", "Dr. L. Hunter"],
    "year": 2025,
    "keywords": ["quantum algorithms", "interstellar navigation", "space travel"]
}


# Compare the two dictionaries using the specified similarity metric
response = client.consensus.distances(
    dict1=dict1,
    dict2=dict2,
    metric="levenshtein_similarity",
)

comparison_results = response.comparison_results
metric_used = response.metric_used

print(f"Metric used: {metric_used}")
print(f"Comparison results: {comparison_results}")
```
