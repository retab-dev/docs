---
title: 'Get Template'
openapi: 'GET /v1/edit/templates/{template_id}'
---

Fetch a single saved edit template, including its full `form_fields` schema.

<RequestExample>

```python python
from retab import Retab

client = Retab()

template = client.edit.templates.get("edittplt_abc123")
print(template.name)
print(template.field_count)
```

```typescript typescript
import { Retab } from '@retab/node';

const client = new Retab();

const template = await client.edit.templates.get("edittplt_abc123");
console.log(template.name);
console.log(template.field_count);
```

```bash curl
curl -X GET \
  'https://api.retab.com/v1/edit/templates/edittplt_abc123' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>
