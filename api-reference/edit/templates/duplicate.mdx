---
title: 'Duplicate Template'
openapi: 'POST /v1/edit/templates/{template_id}/duplicate'
---

Create a copy of an existing template. If `name` is omitted, the API defaults to `"<original> (copy)"`.

<RequestExample>

```python python
from retab import Retab

client = Retab()

duplicate = client.edit.templates.duplicate(
    template_id="edittplt_abc123",
    name="Employee onboarding form copy",
)

print(duplicate.id)
```

```typescript typescript
import { Retab } from '@retab/node';

const client = new Retab();

const duplicate = await client.edit.templates.duplicate("edittplt_abc123", {
    name: "Employee onboarding form copy",
});

console.log(duplicate.id);
```

```bash curl
curl -X POST \
  'https://api.retab.com/v1/edit/templates/edittplt_abc123/duplicate' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Employee onboarding form copy"
  }'
```

</RequestExample>
