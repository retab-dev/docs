---
title: 'Delete Template'
openapi: 'DELETE /v1/edit/templates/{template_id}'
---

Delete a saved edit template.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.edit.templates.delete("edittplt_abc123")
print("Deleted")
```

```typescript typescript
import { Retab } from '@retab/node';

const client = new Retab();

await client.edit.templates.delete("edittplt_abc123");
console.log("Deleted");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/edit/templates/edittplt_abc123' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```text 204
No Content
```

```json 404
{
  "detail": "Template edittplt_missing not found"
}
```

</ResponseExample>
