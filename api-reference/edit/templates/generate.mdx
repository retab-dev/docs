---
title: 'Generate Template'
openapi: 'POST /v1/edit/templates/generate'
---

<RequestExample>

```python python
from retab import Retab
import base64

client = Retab()

# Generate form schema from a PDF document
response = client.edit.templates.generate(
    document="blank_form.pdf",
    model="retab-small"
)

print(f"Detected {response.field_count} fields")

# Access the form schema
for field in response.form_schema.form_fields:
    print(f"Field: {field.key} - {field.description} ({field.type})")

# Save the annotated PDF (with bounding boxes drawn)
if response.annotated_pdf:
    pdf_bytes = base64.b64decode(response.annotated_pdf.url.split(",")[1])
    with open("annotated_form.pdf", "wb") as f:
        f.write(pdf_bytes)
```

```javascript javascript
import { Retab } from '@retab/node';
import fs from 'fs';

const client = new Retab();

// Generate form schema from a PDF document
const response = await client.edit.templates.generate({
    document: "blank_form.pdf",
    model: "retab-small"
});

console.log(`Detected ${response.field_count} fields`);

// Access the form schema
for (const field of response.form_schema.form_fields) {
    console.log(`Field: ${field.key} - ${field.description} (${field.type})`);
}

// Save the annotated PDF
const base64Content = response.annotated_pdf.url.split(",")[1];
fs.writeFileSync("annotated_form.pdf", Buffer.from(base64Content, "base64"));
```

```bash curl
curl -X POST 'https://api.retab.com/v1/edit/templates/generate' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "document": {
      "filename": "blank_form.pdf",
      "url": "data:application/pdf;base64,JVBERi0xLjQK..."
    },
    "model": "retab-small"
  }'
```
