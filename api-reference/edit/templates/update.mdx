---
title: 'Update Template'
openapi: 'PATCH /v1/edit/templates/{template_id}'
---

Update a template name, its saved `form_fields`, or both.

<RequestExample>

```python python
from retab import Retab

client = Retab()

template = client.edit.templates.update(
    template_id="edittplt_abc123",
    name="Employee onboarding form v2",
)

print(template.updated_at)
```

```typescript typescript
import { Retab } from '@retab/node';

const client = new Retab();

const template = await client.edit.templates.update("edittplt_abc123", {
    name: "Employee onboarding form v2",
});

console.log(template.updated_at);
```

```bash curl
curl -X PATCH \
  'https://api.retab.com/v1/edit/templates/edittplt_abc123' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Employee onboarding form v2"
  }'
```
