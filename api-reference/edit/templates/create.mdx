---
title: 'Create Template'
openapi: 'POST /v1/edit/templates'
---

Create a saved PDF edit template from a source document and a list of detected or hand-authored form fields.

<RequestExample>

```python python
from retab import Retab

client = Retab()

schema = client.edit.templates.generate(
    document="blank_form.pdf",
    model="retab-small",
)

template = client.edit.templates.create(
    name="Employee onboarding form",
    document="blank_form.pdf",
    form_fields=schema.form_schema.form_fields,
)

print(template.id)
```

```typescript typescript
import { Retab } from '@retab/node';

const client = new Retab();

const schema = await client.edit.templates.generate({
    document: "blank_form.pdf",
    model: "retab-small",
});

const template = await client.edit.templates.create({
    name: "Employee onboarding form",
    document: "blank_form.pdf",
    form_fields: schema.form_schema.form_fields,
});

console.log(template.id);
```

```bash curl
curl -X POST \
  'https://api.retab.com/v1/edit/templates' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Employee onboarding form",
    "document": {
      "filename": "blank_form.pdf",
      "url": "data:application/pdf;base64,JVBERi0xLjQK..."
    },
    "form_fields": [
      {
        "bbox": {
          "left": 0.14,
          "top": 0.18,
          "width": 0.24,
          "height": 0.03,
          "page": 1
        },
        "description": "Employee full name",
        "type": "text",
        "key": "employee_name",
        "value": null
      }
    ]
  }'
```
