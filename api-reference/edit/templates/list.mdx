---
title: 'List Templates'
openapi: 'GET /v1/edit/templates'
---

List saved edit templates with pagination and optional filtering by template name or MIME type.

<RequestExample>

```python python
from retab import Retab

client = Retab()

templates = client.edit.templates.list(limit=10)
for template in templates.data:
    print(f"{template.id}: {template.name} ({template.field_count} fields)")

filtered = client.edit.templates.list(
    filename="w9",
    mime_type="application/pdf",
    limit=20,
)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const templates = await client.edit.templates.list({ limit: 10 });
for (const template of templates.data) {
    console.log(`${template.id}: ${template.name} (${template.field_count} fields)`);
}

const filtered = await client.edit.templates.list({
    filename: "w9",
    mime_type: "application/pdf",
    limit: 20,
});
```

```bash curl
curl -X GET \
```

## Query Parameters

<ParamField query="before" type="string">
  Cursor for backward pagination.
</ParamField>

<ParamField query="after" type="string">
  Cursor for forward pagination.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Maximum number of templates to return. Range: 1-100.
</ParamField>

<ParamField query="order" type="string" default="desc">
  Sort order. Valid values: `asc`, `desc`.
</ParamField>

<ParamField query="filename" type="string">
  Case-insensitive filter on the template name.
</ParamField>

<ParamField query="mime_type" type="string">
  Filter by the stored template file MIME type. For templates this is typically `application/pdf`.
</ParamField>

<ParamField query="sort_by" type="string" default="created_at">
  Sort field. Supported values: `created_at`, `name`.
</ParamField>
