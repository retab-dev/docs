---
title: 'Fill Document'
openapi: 'POST /v1/edit/agent/fill'
---

Automatically detect fillable regions in a document and apply the values described in `instructions`.
This endpoint supports PDF, DOCX, XLSX, and PPTX inputs.

<RequestExample>

```python python
from retab import Retab

client = Retab()

response = client.edit.agent.fill(
    document="form.pdf",
    instructions="Fill this form with: Name: John Doe, Date: 2025-01-02, Email: john@example.com",
    model="retab-small"
)


# Save the filled document
with open("filled_form.pdf", "wb") as f:
    import base64
    pdf_bytes = base64.b64decode(response.filled_document.url.split(",")[1])
    f.write(pdf_bytes)
```

```javascript javascript
import { Retab } from '@retab/node';
import fs from 'fs';

const client = new Retab();

const response = await client.edit.agent.fill({
    document: "form.pdf",
    instructions: "Fill this form with: Name: John Doe, Date: 2025-01-02",
    model: "retab-small"
});

// Save the filled document
const base64Content = response.filled_document.url.split(",")[1];
fs.writeFileSync("filled_form.pdf", Buffer.from(base64Content, "base64"));
```

```bash curl
curl -X POST 'https://api.retab.com/v1/edit/agent/fill' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "document": {
      "filename": "form.pdf",
      "url": "data:application/pdf;base64,JVBERi0xLjQK..."
    },
    "instructions": "Fill this form with: Name: John Doe, Date: 2025-01-02, Email: john@example.com",
    "model": "retab-small"
  }'
```
