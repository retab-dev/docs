---
title: "Delete Extraction"
openapi: "DELETE /v1/extractions/{extraction_id}"
---

Delete a persisted `Extraction` record by ID. Returns `204 No Content` on success.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

client.extractions.delete("extr_01G34H8J2K")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

await client.extractions.delete("extr_01G34H8J2K");
```

```go Go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Extractions.Delete(ctx, "extr_01G34H8J2K"); err != nil {
		log.Fatal(err)
	}
}
```

```ruby Ruby
require 'retab'
```

