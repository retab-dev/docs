---
title: 'Delete Extraction'
openapi: 'DELETE /v1/extractions/{extraction_id}'
---

Delete a persisted `Extraction` record by ID. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.extractions.delete("extr_01G34H8J2K")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

await client.extractions.delete("extr_01G34H8J2K");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/extractions/extr_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```text 204
No Content
```

```json 404
{
  "detail": "Extraction not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="extraction_id" type="string" required>
  The unique identifier of the extraction to delete.
</ParamField>
