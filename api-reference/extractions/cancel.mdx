---
title: "Cancel Extraction"
openapi: "POST /v1/extractions/{extraction_id}/cancel"
---

Cancel an in-flight background `Extraction` run (one created with `background: true`). Cancellation is idempotent: a run that has already reached a terminal state (`completed`, `failed`, or `cancelled`) is returned unchanged rather than erroring.

<RequestExample>

```curl cURL
curl -X POST \
  'https://api.retab.com/v1/extractions/extr_01G34H8J2K/cancel' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```json 200
{
  "id": "extr_01G34H8J2K",
  "status": "cancelled",
  "error": null
}
```

```json 404
{
  "detail": "extraction 'extr_01G34H8J2K' not found"
}
```

</ResponseExample>
