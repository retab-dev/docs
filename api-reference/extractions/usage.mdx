---
title: 'Get Extraction Usage'
openapi: 'GET /v1/extractions/{extraction_id}/usage'
---

Return the LLM-token usage breakdown for a single extraction (input tokens,
output tokens, total tokens, and the cost recorded at extraction time).

<Note>
There is no dedicated SDK helper for this endpoint. Use the Python or Node
HTTP client to fetch it directly, or fall back to curl below.
</Note>

<RequestExample>

```python python
import httpx
import os

response = httpx.get(
    "https://api.retab.com/v1/extractions/extr_01G34H8J2K/usage",
    headers={"Api-Key": os.environ["RETAB_API_KEY"]},
)
response.raise_for_status()
print(response.json())
```

```typescript typescript
const response = await fetch(
  "https://api.retab.com/v1/extractions/extr_01G34H8J2K/usage",
  { headers: { "Api-Key": process.env.RETAB_API_KEY! } },
);
const usage = await response.json();
console.log(usage);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/extractions/extr_01G34H8J2K/usage' \
  -H 'accept: application/json' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "extraction_id": "extr_01G34H8J2K",
  "input_tokens": 4128,
  "output_tokens": 521,

```


</ResponseExample>
