---
title: 'Generate'
openapi: 'POST /v1/schemas/generate'
---




<RequestExample>

```python python
from retab import Retab
client = Retab()

response = client.schemas.generate(
    documents=["passport.jpeg"],
    model="retab-small",      # or any model your plan supports
    temperature=0.5,          #  put 0.5 for more creative schema
    image_resolution_dpi=96,
)

print("Generated JSON Schema:")
print(response)
```

```javascript javascript
import { Retab } from '@retab/node';
import { config } from 'dotenv';

config();

const client = new Retab();

const result = await client.schemas.generate({
    documents: ["passport.jpeg"],
    model: "retab-small",
    temperature: 0.5, // put 0.5 for more creative schema generation
    image_resolution_dpi: 96,
});
```


```bash curl
curl https://api.retab.com/v1/schemas/generate \
  -H "Api-Key: $RETAB_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "documents": [
      {
        "filename": "passport.jpeg",
        "url": "data:image/jpeg;base64,iVBORw0KGgoAAAANSUhEUgAAADIA..."
      }
    ],
    "model": "retab-small",
    "temperature": 0.5,
    "image_resolution_dpi": 96
  }'

```
