---
title: "Get Table"
openapi: "GET /v1/tables/{table_id}"
---

Get one table's metadata, schema, and sample rows.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

table = client.tables.get(table_id="workflow_table_123")
print(table)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const table = await client.tables.get("workflow_table_123");
console.log(table);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	table, err := client.Tables.Get(ctx, "workflow_table_123")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(*table)
}
```
