---
title: "Download Table CSV"
openapi: "GET /v1/tables/{table_id}/download"
---

Download the CSV backing a table.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

csv_bytes = client.tables.download(table_id="workflow_table_123")
print(len(csv_bytes))
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const csv = await client.tables.download("workflow_table_123");
console.log(csv.size);
```

```go Go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Tables.Download(ctx, "workflow_table_123"); err != nil {
		log.Fatal(err)
	}
}
```

```rust Rust
use retab::Retab;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Retab::new(std::env::var("RETAB_API_KEY")?);

    let csv = client.tables().download("workflow_table_123").await?;
    println!("{} bytes", csv.len());
    Ok(())
}
```
