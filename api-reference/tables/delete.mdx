---
title: "Delete Table"
openapi: "DELETE /v1/tables/{table_id}"
---

Delete a table and its stored CSV snapshot.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

client.tables.delete(table_id="workflow_table_123")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

await client.tables.delete("workflow_table_123");
```

```go Go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Tables.Delete(ctx, "workflow_table_123"); err != nil {
		log.Fatal(err)
	}
}
```
