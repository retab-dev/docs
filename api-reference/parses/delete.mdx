---
title: "Delete Parse"
openapi: "DELETE /v1/parses/{parse_id}"
---

Permanently delete a persisted parse. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.parses.delete("parse_01G34H8J2K")
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

await client.parses.delete("parse_01G34H8J2K");
```

```go go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Parses.Delete(ctx, "parse_01G34H8J2K"); err != nil {
		log.Fatal(err)
	}
}
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/parses/parse_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>
