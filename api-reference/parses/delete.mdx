---
title: 'Delete Parse'
openapi: 'DELETE /v1/parses/{parse_id}'
---

Permanently delete a persisted parse. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.parses.delete("parse_01G34H8J2K")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

await client.parses.delete("parse_01G34H8J2K");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/parses/parse_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```json 204
```

```json 404
{
  "detail": "Parse not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="parse_id" type="string" required>
  The ID of the parse to delete.
</ParamField>
