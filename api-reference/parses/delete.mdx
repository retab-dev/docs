---
title: "Delete Parse"
openapi: "DELETE /v1/parses/{parse_id}"
---

Delete a parse by ID. The deleted parse no longer appears in list results or retrieval calls.

<RequestExample>

```python Python
from retab import Retab

client = Retab()
client.parses.delete("parse_01G34H8J2K")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });
await client.parses.delete("parse_01G34H8J2K");
```

```go Go

	}

		log.Fatal(err)
	}
}
```
