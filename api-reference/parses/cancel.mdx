---
title: "Cancel Parse"
openapi: "POST /v1/parses/{parse_id}/cancel"
---

Cancel an in-flight background `Parse` run (one created with `background: true`). Cancellation is idempotent: a run that has already reached a terminal state (`completed`, `failed`, or `cancelled`) is returned unchanged rather than erroring.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

parse = client.parses.cancel("prs_01G34H8J2K")
print(parse.status)
```

```typescript TypeScript

console.log(parse.status);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	parse, err := client.Parses.Cancel(context.Background(), "prs_01G34H8J2K")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(parse.Status)
}
```
