---
title: 'List Parses'
openapi: 'GET /v1/parses'
---

List persisted parses for your organization, with cursor-based pagination and optional filename / date filtering.

<RequestExample>

```python python
from datetime import datetime
from retab import Retab

client = Retab()

# List recent parses
parses = client.parses.list(limit=20, order="desc")
for parse in parses.data:
    print(f"{parse.id}: {parse.file.filename}")

# Filter by date range
parses = client.parses.list(
    from_date=datetime(2024, 1, 1),
    to_date=datetime(2024, 12, 31),
    limit=50,
)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

// List recent parses
const parses = await client.parses.list({ limit: 20, order: "desc" });
for (const parse of parses.data) {
    console.log(`${parse.id}: ${parse.file.filename}`);
}

// Filter by date range
const ranged = await client.parses.list({
    from_date: new Date("2024-01-01"),
    to_date: new Date("2024-12-31"),
    limit: 50,
});
```

```bash curl
# List recent parses
curl -X GET \
  'https://api.retab.com/v1/parses?limit=20&order=desc' \
  -H "Api-Key: $RETAB_API_KEY"

# Filter by filename prefix and date range
curl -X GET \
  'https://api.retab.com/v1/parses?filename=invoice&from_date=2024-01-01&to_date=2024-12-31&limit=50' \
  -H "Api-Key: $RETAB_API_KEY"
```

## Query Parameters

<ParamField query="before" type="string">
  Cursor for backward pagination.
</ParamField>

<ParamField query="after" type="string">
  Cursor for forward pagination.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Maximum number of parses to return.
</ParamField>

<ParamField query="order" type="string" default="desc">
  Sort order by `created_at`. Valid values: `asc`, `desc`.
</ParamField>

<ParamField query="filename" type="string">
  Case-insensitive substring filter applied to `file.filename`.
</ParamField>

<ParamField query="from_date" type="string">
  Return parses created on or after this date, in `YYYY-MM-DD` format.
</ParamField>

<ParamField query="to_date" type="string">
  Return parses created on or before this date, in `YYYY-MM-DD` format.
</ParamField>

## Count

To fetch only the number of parses matching a filter (without paginating through results), use:

```bash
GET /v1/parses/count?filename=invoice&from_date=2024-01-01
```

which returns `{ "count": <int> }`. The supported query parameters are the same as for list, minus pagination (`before`, `after`, `limit`, `order`).
