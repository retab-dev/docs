---
title: 'Get Parse'
openapi: 'GET /v1/parses/{parse_id}'
---

Retrieve a single persisted parse by ID.

<RequestExample>

```python python
from retab import Retab

client = Retab()

parse = client.parses.get("parse_01G34H8J2K")
print(parse.output.text)
print(parse.file.filename)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const parse = await client.parses.get("parse_01G34H8J2K");
console.log(parse.output.text);
console.log(parse.file.filename);
```

```bash curl
curl -X GET \
  'https://api.retab.com/v1/parses/parse_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>
