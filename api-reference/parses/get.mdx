---
title: "Get Parse"
openapi: "GET /v1/parses/{parse_id}"
---

Retrieve a single persisted parse by ID.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

parse = client.parses.get("parse_01G34H8J2K")
print(parse.output.text)
print(parse.file.filename)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const parse = await client.parses.get("parse_01G34H8J2K");
console.log(parse.output.text);
console.log(parse.file.filename);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	parse, err := client.Parses.Get(ctx, "parse_01G34H8J2K")
	if err != nil {
		log.Fatal(err)
	}
	output, _ := (*parse)["output"].(map[string]any)
	fmt.Println(output["text"])
	fmt.Println((*parse)["file"])
}
```

```rust Rust
use retab::Retab;

}
```
