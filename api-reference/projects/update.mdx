title: 'Update'
openapi: 'PATCH /v1/projects/{project_id}'

<RequestExample>
```python python
from retab import Retab

client = Retab()
response = client.projects.update("<project_id>", name="New name")
```

```javascript javascript
import { Retab } from '@retab/node';
import { config } from 'dotenv';

config();

const client = new Retab();

const response = await client.projects.update("<project_id>", {
    name: "New name",
});
```

```bash curl
curl --request PATCH \
  -H "API-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  --url https://api.retab.com/v1/projects/{project_id} \
  --data '{
  "name": "Invoices"
}'
```
