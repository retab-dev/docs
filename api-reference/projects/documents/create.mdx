title: 'Create'
openapi: 'POST /v1/projects/{project_id}/documents'
---

<RequestExample>

```python python
from retab import Retab

client = Retab()
response = client.projects.documents.create(
    project_id="<project_id>",
    document="Invoice.pdf",
    annotation={
        "invoice_number": "12345"
    }
)
```

```javascript javascript
import { Retab } from '@retab/node';
import { config } from 'dotenv';

config();

const client = new Retab();

const response = await client.projects.documents.create(
    '<project_id>',
    {
        mime_data: "Invoice.pdf",
        annotation: {
            invoice_number: '12345'
        }
    }
);
```

```bash curl
curl --request POST \
  --url https://api.retab.com/v1/projects/{project_id}/documents \
  --header 'Content-Type: application/json' \
  --data '{
  "mime_data": {
    "filename": "file.pdf",
    "url": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAADIA..."
  },
  "annotation": {}
}'
```
