---
title: 'Update'
openapi: 'PATCH /v1/projects/{project_id}/documents/{document_id}'
---

<RequestExample>
```python python
from retab import Retab

client = Retab()
response = client.projects.documents.update("<project_id>", "<document_id>", annotation={"invoice_number": "54321"})
```

```javascript javascript
import { Retab } from '@retab/node';
import { config } from 'dotenv';

config();

const client = new Retab();

const response = await client.projects.documents.update('<project_id>', '<document_id>', {
    annotation: {
        invoice_number: '54321'
    }
});
```

```bash curl
curl --request PATCH \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  --url https://api.retab.com/v1/projects/{project_id}/documents/{document_id} \
  --data '{
  "annotation": {
    "invoice_number": "54321"
  }
}'
```
