---
title: 'Export Workflow Spec'
openapi: 'GET /v1/workflows/spec/{workflow_id}'
---

Export the current draft workflow as canonical declarative YAML.

Exports use `apiVersion: workflows.retab.com/v1alpha2` and include raw explicit handles on every non-synthetic edge.

<RequestExample>

```python python
from retab import Retab

client = Retab()

exported = client.workflows.specs.export("wf_abc123")

print(exported.yaml_definition)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const exported = await client.workflows.specs.export("wf_abc123");

console.log(exported.yaml_definition);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/spec/wf_abc123' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "workflow_id": "wf_abc123",
  "yaml_definition": "apiVersion: workflows.retab.com/v1alpha2\nkind: Workflow\nmetadata:\n  id: wf_abc123\n  name: Invoice Workflow\nspec:\n  blocks:\n    start-node:\n      type: start\n      label: Input Document\n  edges: []\n"
}
```

</ResponseExample>
