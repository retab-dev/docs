---
title: 'Apply Workflow Spec'
openapi: 'POST /v1/workflows/yaml/apply'
---

Apply a declarative workflow YAML spec to draft workflow state.

`apply()` writes the canonical draft graph represented by the YAML spec. Pass `publish=True` in Python, or `{ publish: true }` in JavaScript, to publish immediately after applying.

<RequestExample>

```python python
from retab import Retab

client = Retab()

result = client.workflows.specs.apply(
    yaml_definition,
    publish=False,
)

print(result.workflow_id)
print(result.created)
print(result.published)
```

```javascript javascript
import { Retab } from '/node';

const client = new Retab();

const result = await client.workflows.specs.apply(yamlDefinition, {
  publish: false,
});

console.log(result.workflow_id);
console.log(result.created);
console.log(result.published);
```

```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/yaml/apply' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "yaml_definition": "<workflow-yaml>",
    "publish": false
  }'
```

</RequestExample>
