---
title: "Get Version"
openapi: "GET /v1/workflows/reviews/versions/{rvr_id}"
---

Fetch one review version by its content-addressed `rvr_…` id.

<RequestExample>

```python python
version = client.workflows.reviews.versions.get(
    "rvr_PFBGO3R6T3PG5O37U4Y3NDXWAM",
)
print(version.snapshot)
```

```javascript javascript
const version = await client.workflows.reviews.versions.get(
  "rvr_PFBGO3R6T3PG5O37U4Y3NDXWAM",
);
```

```bash curl
curl -X GET \
  'https://api.retab.com/v1/workflows/reviews/versions/rvr_PFBGO3R6T3PG5O37U4Y3NDXWAM' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
