---
title: "Get Version"
openapi: "GET /v1/workflows/reviews/versions/{version_id}"
---

Fetch one review version by its content-addressed `rvr_…` id.

<RequestExample>

```python python
from retab import Retab

client = Retab()

version = client.workflows.reviews.versions.get(
    "rvr_PFBGO3R6T3PG5O37U4Y3NDXWAM",
)
print(version.snapshot)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const version = await client.workflows.reviews.versions.get(
  "rvr_PFBGO3R6T3PG5O37U4Y3NDXWAM",
);

console.log(version.snapshot);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	version, err := client.Workflows.Reviews.Versions.Get(ctx, "rvr_PFBGO3R6T3PG5O37U4Y3NDXWAM")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(version.Snapshot)
}
```

```bash curl

  -H 'Api-Key: <your-api-key>'
```
