---
title: 'Submit Decision'
openapi: 'POST /v1/workflows/reviews/{run_id}/{block_id}/decision'
---

Submit a terminal verdict against one exact review version.

There are two verdicts:

| Verdict | Effect |
|---|---|
| `approved` | The selected `version_id` flows downstream and the run resumes. |
| `rejected` | The run is cancelled. `reason` is required. |

The request body carries:

| Field | Notes |
|---|---|
| `verdict` | `approved` / `rejected`. |
| `version_id` | Exact key from `versions_by_id`. |
| `reason` | Required for `rejected`. |

The SDK exposes one method per verdict — `approve` and `reject` — both of which post to this endpoint.

The response includes:

| Field | Notes |
|---|---|
| `submission_status` | Whether this request created the decision or matched an existing final decision. |
| `review` | The current review, including `versions_by_id` and `decision`. |
| `resume_status` | `resumed`, `skipped`, or `failed` for the runtime resume/cancel signal. |
| `resume_error` | Present when the decision was committed but runtime resume/cancel signaling failed. |

`submission_status` values:

| Status | Meaning |
|---|---|
| `accepted` | The terminal decision was created by this request. |
| `already_applied` | The review already had the exact same verdict and `version_id`; the server returned the current review without creating another decision. |

<RequestExample>

```python python
from retab import Retab

client = Retab()

review = client.workflows.reviews.get("run_abc123", "block_extract")
version_id = next(iter(review.versions_by_id))

result = client.workflows.reviews.approve(
    "run_abc123",
    "block_extract",
    version_id=version_id,
)
print(result.submission_status)

client.workflows.reviews.reject(
    "run_abc123",
    "block_extract",
    version_id=version_id,
    reason="Source document is illegible; cannot verify totals.",
)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const review = await client.workflows.reviews.get("run_abc123", "block_extract");
const versionId = Object.keys(review.versions_by_id)[0];

const result = await client.workflows.reviews.approve("run_abc123", "block_extract", {
  versionId,
});
console.log(result.submission_status);

await client.workflows.reviews.reject("run_abc123", "block_extract", {
  versionId,
  reason: "Source document is illegible; cannot verify totals.",
});
```

```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/reviews/run_abc123/block_extract/decision' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "verdict": "approved",
    "version_id": "b7d7f3ddc4ce98edbb5ea71d2ff8a7b711b3d4b70c20f481e14c03c7d53e15e2"
  }'
```
