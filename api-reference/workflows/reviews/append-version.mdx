---
title: 'Append Version'
openapi: 'POST /v1/workflows/reviews/{run_id}/{block_id}/versions'
---

Append a corrective edit version to a review.

This records a new immutable snapshot of the gated block's output without submitting a verdict. The new version is addressed by the content hash of its `snapshot`.

The request body carries:

| Field | Notes |
|---|---|
| `snapshot` | The full corrected output payload. It must match the gated block's canonical review snapshot shape. |
| `parent_id` | The content-hash id of the version this snapshot was edited from. |
| `origin` | Provenance label for the version, e.g. `"human_edit"` (default). |
| `note` | Optional free-text note explaining the correction. |

Snapshot validation is block-specific:

| Block type | Required snapshot |
|---|---|
| `extract` | Raw extracted JSON object. |
| `classifier` | `{ "category": "..." }` only. Reasoning is not review state. |
| `split` | `{ "documents": [{ "name": "...", "pages": [1, 2] }] }`. |
| `for_each` | `{ "partitions": [{ "key": "...", "pages": [1, 2] }] }`. Only split-by-key for-each reviews are supported. |

For split and for-each snapshots, page arrays must contain positive integers, sorted ascending, with no duplicates. Invalid snapshots return `422` and are not stored.

The API does not support partial patches. Always submit the complete replacement snapshot you want reviewers to decide on.

<RequestExample>

```python python
from retab import Retab

client = Retab()

review = client.workflows.reviews.get("run_abc123", "block_extract")
parent_id = next(iter(review.versions_by_id))

review = client.workflows.reviews.edit(
    "run_abc123",
    "block_extract",
    parent_id=parent_id,
    snapshot={"total_amount": 325, "vendor_name": "Acme Corp"},
    origin="human_edit",
    note="Corrected total.",
)
print("versions:", len(review.versions_by_id))
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

let review = await client.workflows.reviews.get("run_abc123", "block_extract");
const parentId = Object.keys(review.versions_by_id)[0];

review = await client.workflows.reviews.edit("run_abc123", "block_extract", {
  parentId,
  snapshot: { total_amount: 325, vendor_name: "Acme Corp" },
  origin: "human_edit",
  note: "Corrected total.",
});
console.log("versions:", Object.keys(review.versions_by_id).length);
```

```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/reviews/run_abc123/block_extract/versions' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "snapshot": { "total_amount": 325, "vendor_name": "Acme Corp" },
    "parent_id": "a9f4d2e7f50b8b7bc84f4d96d2ad831a2a3bfead9b46f945baf2f8eb2f457a18",
    "origin": "human_edit",
    "note": "Corrected total."
  }'
```
