---
title: "Append Version"
openapi: "POST /v1/workflows/reviews/{id}/versions"
---

Append a complete replacement snapshot to one review without deciding it.

The response includes `append_status`, the new or existing `version_id`, and the updated `review`.

<RequestExample>

```python python
appended = client.workflows.reviews.append_version(
    "rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY",
    parent_version_id="rvr_HQ6ZZBQTJLSGLZQCWBSI25SCOQ",
    snapshot={"total_amount": 325, "vendor_name": "Acme Corp"},
    note="Corrected total.",
)
print(appended.version_id)
```

```javascript javascript
const appended = await client.workflows.reviews.appendVersion(
  "rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY",
  {
    parentVersionId: "rvr_HQ6ZZBQTJLSGLZQCWBSI25SCOQ",
    snapshot: { total_amount: 325, vendor_name: "Acme Corp" },
    note: "Corrected total.",
  },
);
```

```bash curl
curl -X POST \
  'https://api.retab.com/v1/workflows/reviews/rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY/versions' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "parent_version_id": "rvr_HQ6ZZBQTJLSGLZQCWBSI25SCOQ",
    "snapshot": { "total_amount": 325, "vendor_name": "Acme Corp" },
    "note": "Corrected total."
  }'
```

</RequestExample>
