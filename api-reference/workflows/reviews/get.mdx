---
title: 'Get Review'
openapi: 'GET /v1/workflows/reviews/{run_id}/{block_id}'
---

Get the full review overlay for a gated block run.

The overlay is a versioned sidecar attached to a single `(run_id, block_id)` pair. It records:

| Field | Notes |
|---|---|
| `versions[]` | Every version of the gated block's output. Sequence `0` is the model's original; later sequences are corrective edits. |
| `decisions[]` | Every verdict submitted against the overlay (`approved` / `rejected` / `escalated`). |
| `audit[]` | The append-only audit trail — every claim, release, edit, and decision, with the acting `Actor`. |
| `rev` | The overlay's compare-and-swap token. Read it here, pass it back (as the `version_stamp` request parameter) on every mutating call. |
| `claim` | The `ReviewClaim` currently held on the overlay (`{holder, claimed_at, expires_at}`), or `null`. |

Every actor — model, agent, or human — is described by the same symmetric `Actor` shape: `{kind, id, display_name}`.

<Note>
The `rev` you read here is what every mutating call ([append version](/api-reference/workflows/reviews/append-version), [submit decision](/api-reference/workflows/reviews/submit-decision), [claim](/api-reference/workflows/reviews/claim), [release](/api-reference/workflows/reviews/release)) must echo back. If it has advanced since you read it, the mutating call returns `409` — re-read and retry.
</Note>

<RequestExample>

```python python
from retab import Retab

client = Retab()

overlay = client.workflows.runs.reviews.get("run_abc123", "hil-1")

print(overlay.status, overlay.rev)

# Version 0 is always the model's original output
original = overlay.versions[0]
print(original.origin, original.author.kind)

# The effective output is the latest version
latest = overlay.versions[-1]
print(latest.snapshot)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const overlay = await client.workflows.runs.reviews.get("run_abc123", "hil-1");

console.log(overlay.status, overlay.rev);

// Version 0 is always the model's original output
const original = overlay.versions[0];
console.log(original.origin, original.author.kind);

// The effective output is the latest version
const latest = overlay.versions[overlay.versions.length - 1];
console.log(latest.snapshot);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/reviews/run_abc123/hil-1' \
  -H 'Api-Key: <your-api-key>'
```
