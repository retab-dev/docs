---
title: "Get Review"
openapi: "GET /v1/workflows/reviews/{id}"
---

Get one full review by opaque review id.

<RequestExample>

```python python
review = client.workflows.reviews.get("rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY")

for version_id, version in review.versions.items():
    print(version_id, version.parent_id, version.author.kind)
```

```javascript javascript
const review = await client.workflows.reviews.get(
  "rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY",
);

for (const [versionId, version] of Object.entries(review.versions)) {
  console.log(versionId, version.parent_id, version.author.kind);
}
```

```bash curl
curl -X GET \
  'https://api.retab.com/v1/workflows/reviews/rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
