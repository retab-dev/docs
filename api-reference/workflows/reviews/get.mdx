---
title: 'Get Review'
openapi: 'GET /v1/workflows/reviews/{run_id}/{block_id}'
---

Get the full review for a gated block run.

The review is attached to a single `(run_id, block_id)` pair. It records immutable output snapshots in `versions_by_id`, keyed by content hash, and a single terminal `decision` once the review is resolved.

| Field | Notes |
|---|---|
| `versions_by_id` | Output snapshots keyed by `sha256(canonical_json(snapshot))`. |
| `decision` | `null` while awaiting review; otherwise the terminal `approved` / `rejected` verdict. |
| `triggered_by` | The review predicate that opened the gate. |
| `awaiting_since` | When the block began waiting for review. |

Every actor — model, agent, or human — is described by the same symmetric `Actor` shape: `{kind, id, display_name}`.

<RequestExample>

```python python
from retab import Retab

client = Retab()

review = client.workflows.reviews.get("run_abc123", "block_extract")

for version_id, version in review.versions_by_id.items():
    print(version_id, version.parent_id, version.author.kind)

print("decision:", review.decision)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const review = await client.workflows.reviews.get("run_abc123", "block_extract");

for (const [versionId, version] of Object.entries(review.versions_by_id)) {
  console.log(versionId, version.parent_id, version.author.kind);
}

console.log("decision:", review.decision);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/reviews/run_abc123/block_extract' \
  -H 'Api-Key: <your-api-key>'
```
