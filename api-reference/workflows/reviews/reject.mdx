---
title: "Reject Review"
openapi: "POST /v1/workflows/reviews/{id}/reject"
---

Reject one exact review version. The workflow run is cancelled and `reason` is required.

<RequestExample>

```python python
result = client.workflows.reviews.reject(
    "rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY",
    version_id="ver_HQ6ZZBQTJLSGLZQCWBSI25SCOQ",
    reason="Source document is illegible; cannot verify totals.",
)
```

```javascript javascript
const result = await client.workflows.reviews.reject(
  "rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY",
  {
    versionId: "ver_HQ6ZZBQTJLSGLZQCWBSI25SCOQ",
    reason: "Source document is illegible; cannot verify totals.",
  },
);
```

```bash curl
curl -X POST \
  'https://api.retab.com/v1/workflows/reviews/rev_D4J7WZBRV4H7C2SKQJ4NP6W2EY/reject' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "version_id": "ver_HQ6ZZBQTJLSGLZQCWBSI25SCOQ",
    "reason": "Source document is illegible; cannot verify totals."
  }'
```

</RequestExample>
