---
title: "Get Experiment"
openapi: "GET /v1/workflows/experiments/{experiment_id}?workflow_id={workflow_id}"
---

Fetch one experiment by id. The response includes the latest run status,
score, staleness, and schema-drift information — the same shape as the
listing endpoint.

<RequestExample>

```python python
from retab import Retab

client = Retab()

experiment = client.workflows.experiments.get(
    workflow_id="wf_abc123",
    experiment_id="exp_abc",
)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const experiment = await client.workflows.experiments.get(
  "wf_abc123",
  "exp_abc",
);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	experiment, err := client.Workflows.Experiments.Get(ctx, "wf_abc123", "exp_abc")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(experiment.ID)
}
```

```bash curl
curl -X 'GET' \

  -H 'Api-Key: <your-api-key>'
```
