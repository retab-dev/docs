---
title: "Get Experiment Run Result"
openapi: "GET /v1/workflows/experiments/results/{result_id}"
---

Fetch one experiment result by its opaque `result_id`. Use this to refresh a
single document result without re-listing every result for the parent
experiment run.

<RequestExample>

```python python
from retab import Retab

client = Retab()

result = client.workflows.experiments.runs.results.get("expresult_1")

print(result.document_id)
print(result.lifecycle.status)
print(result.artifact)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const result = await client.workflows.experiments.runs.results.get({
  resultId: "expresult_1",
});
console.log(result.document_id);
console.log(result.lifecycle.status);
console.log(result.artifact);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	result, err := client.Workflows.Experiments.Runs.Results.Get(ctx, "expresult_1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(result.DocumentID)
	fmt.Println(result.Lifecycle.Status)
	fmt.Println(result.Artifact)
}
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/experiments/results/expresult_1' \
  -H 'Api-Key: <your-api-key>'
```
