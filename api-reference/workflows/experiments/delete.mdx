---
title: "Delete Experiment"
openapi: "DELETE /v1/workflows/experiments/{experiment_id}"
---

Permanently delete an experiment along with all its runs and per-document
results.

Returns `409` if any run is still pending or running. Cancel the in-flight run
with [Cancel Experiment Run](/api-reference/workflows/experiments/runs/cancel)
first.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.workflows.experiments.delete(
    experiment_id="exp_abc",
)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

await client.workflows.experiments.delete("exp_abc");
```

```go go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Workflows.Experiments.Delete(ctx, "exp_abc"); err != nil {
		log.Fatal(err)
	}
}
```

```bash curl
curl -X 'DELETE' \
  'https://api.retab.com/v1/workflows/experiments/exp_abc' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
