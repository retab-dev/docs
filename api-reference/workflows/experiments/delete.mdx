---
title: 'Delete Experiment'
openapi: 'DELETE /v1/workflows/{workflow_id}/experiments/{experiment_id}'
---

Permanently delete an experiment along with all its runs and per-document
jobs.

Returns `409` if the latest run is still pending or running — call
[Cancel Experiment](/api-reference/workflows/experiments/cancel) first.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.workflows.experiments.delete(
    workflow_id="wf_abc123",
    experiment_id="exp_abc",
)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

await client.workflows.experiments.delete("wf_abc123", "exp_abc");
```

```go go
package main

import (
	"io"
	"log"
	"net/http"
	"os"
)

func main() {
	request, err := http.NewRequest(
		http.MethodDelete,
		"https://api.retab.com/v1/workflows/wf_abc123/experiments/exp_abc",
		nil,
	)
	if err != nil {
		log.Fatal(err)
	}
	request.Header.Set("Api-Key", os.Getenv("RETAB_API_KEY"))

	response, err := http.DefaultClient.Do(request)
	if err != nil {
		log.Fatal(err)
	}
	defer response.Body.Close()

	responseBody, err := io.ReadAll(response.Body)
	if err != nil {
		log.Fatal(err)
	}
	if response.StatusCode >= 400 {
		log.Fatalf("request failed: %s", responseBody)
	}
}
```
