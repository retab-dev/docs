---
title: 'Get Experiment Job'
openapi: 'GET /v1/workflows/{workflow_id}/experiments/{experiment_id}/runs/{run_id}/documents/{document_id}'
---

Get one document's execution job inside a specific experiment run.

Use this to inspect a single document's outcome — its handle inputs, the
canonical artifact the block produced, and any error message. For the full
run's content (one job per document), use
[Get Experiment Content](/api-reference/workflows/experiments/get-content).

<RequestExample>

```python python
from retab import Retab

client = Retab()

job = client.workflows.experiments.runs.get_job(
    workflow_id="wf_abc123",
    experiment_id="exp_abc",
    run_id="exprun_1",
    document_id="expdoc_xyz",
)
print(job.status, job.artifact, job.error)
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	job, err := client.Workflows.Experiments.Runs.GetJob(ctx,
		"wf_abc123", "exp_abc", "exprun_1", "expdoc_xyz")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(job.Status, job.Artifact, job.Error)
}
```
