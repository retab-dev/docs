---
title: 'List Experiment Runs'
openapi: 'GET /v1/workflows/{workflow_id}/experiments/{experiment_id}/runs'
---

List historical runs for one experiment, newest first. Each row is a slim
projection of an `ExperimentRun` (status, score, fingerprints, timing) — the
fields needed for a runs table. For the underlying per-document execution data,
use [Get Experiment Content](/api-reference/workflows/experiments/get-content);
for consensus scores, use
[Get Experiment Metrics](/api-reference/workflows/experiments/get-metrics).

<RequestExample>

```python python
from retab import Retab

client = Retab()

result = client.workflows.experiments.runs.list(
    workflow_id="wf_abc123",
    experiment_id="exp_abc",
)
for run in result.runs:
    print(run.id, run.status, run.score, run.created_at)
```


```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	result, err := client.Workflows.Experiments.Runs.List(ctx, "wf_abc123", "exp_abc")
	if err != nil {
		log.Fatal(err)
	}

	for _, run := range result.Runs {
		fmt.Println(run.ID, run.Status, run.Score, run.CreatedAt)
	}
}
```
