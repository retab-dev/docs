---
title: "Get Experiment Run Result"
openapi: "GET /v1/workflows/experiments/runs/{run_id}/results/{document_id}"
---

Fetch one per-document result from an experiment run. The child key is
`document_id`, not an internal job or result id.

<RequestExample>

```python python
from retab import Retab

client = Retab()

result = client.workflows.experiments.runs.results.get(
    "exprun_2",
    "expdoc_xyz",
)
print(result.lifecycle.status, result.artifact)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const result = await client.workflows.experiments.runs.results.get(
  "exprun_2",
  "expdoc_xyz",
);
console.log(result.lifecycle.status, result.artifact);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/experiments/runs/exprun_2/results/expdoc_xyz' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
