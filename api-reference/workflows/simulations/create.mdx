---
title: "Create Workflow Simulation"
openapi: "POST /v1/workflows/simulations"
---

Create a workflow block simulation from a prior run.

<RequestExample>

```python python
from retab import Retab

client = Retab()

simulation = client.workflows.simulations.create(
    run_id="run_abc123xyz",
    block_id="blk_extract_1",
    step_id="step_iter_0_blk_extract_1",
    n_consensus=5,
)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const simulation = await client.workflows.simulations.create({
  runId: "run_abc123xyz",
  blockId: "blk_extract_1",
  stepId: "step_iter_0_blk_extract_1",
  nConsensus: 5,
});
```

```go go
simulation, err := client.Workflows.Simulations.Create(ctx, retab.CreateWorkflowSimulationRequest{
	RunID:        "run_abc123xyz",
	BlockID:      "blk_extract_1",
	StepID:       "step_iter_0_blk_extract_1",
	NConsensus:   5,
})
```

```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/simulations' \
  -H 'Api-Key: <your-api-key>' \
  -H 'Content-Type: application/json' \
  -d '{"run_id":"run_abc123xyz","block_id":"blk_extract_1"}'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "id": "wfsim_abc123",
  "run_id": "run_abc123xyz",
  "block_id": "blk_extract_1",
  "success": true
}
```

</ResponseExample>
