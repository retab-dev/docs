---
title: "List Workflow Simulations"
openapi: "GET /v1/workflows/simulations"
---

List workflow block simulations for one run and block.

<RequestExample>

```python python
from retab import Retab

client = Retab()

simulations = client.workflows.simulations.list(
    run_id="run_abc123xyz",
    block_id="blk_extract_1",
)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const simulations = await client.workflows.simulations.list({
  runId: "run_abc123xyz",
  blockId: "blk_extract_1",
});
```

```go go
simulations, err := client.Workflows.Simulations.List(ctx, retab.ListWorkflowSimulationsParams{
	RunID:   "run_abc123xyz",
	BlockID: "blk_extract_1",
	Limit:   20,
})
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/simulations?run_id=run_abc123xyz&block_id=blk_extract_1' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
