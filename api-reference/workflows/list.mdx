---
title: 'List Workflows'
openapi: 'GET /v1/workflows'
---

List workflows with cursor pagination. Use `before` or `after` to page through results, and `fields` to limit the returned fields.

<RequestExample>

```python python
from retab import Retab

client = Retab()

workflows = client.workflows.list(
    limit=10,
    order="desc",
    sort_by="updated_at",
)

filtered = client.workflows.list(
    after="workflow_abc123xyz",
    limit=5,
    fields="id,name,updated_at",
)

print(workflows.list_metadata.after)
print(filtered.data[0]["name"])
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const workflows = await client.workflows.list({
  limit: 10,
  order: "desc",
  sortBy: "updated_at",
});

const filtered = await client.workflows.list({
  after: "workflow_abc123xyz",
  limit: 5,
  fields: "id,name,updated_at",
});

console.log(workflows.list_metadata.after);
console.log(filtered.data[0].name);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows?limit=10&order=desc&sort_by=updated_at' \
  -H 'accept: application/json' \
  -H 'Api-Key: <your-api-key>'

curl -X 'GET' \
  'https://api.retab.com/v1/workflows?after=workflow_abc123xyz&limit=5&fields=id%2Cname%2Cupdated_at' \
  -H 'accept: application/json' \
  -H 'Api-Key: <your-api-key>'
```
