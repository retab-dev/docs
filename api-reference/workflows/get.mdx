---
title: 'Get Workflow'
openapi: 'GET /v1/workflows/{workflow_id}'
---

Get a single workflow by ID.

<RequestExample>

```python python
from retab import Retab

client = Retab()

workflow = client.workflows.get("workflow_abc123xyz")

print(workflow.name)
print(workflow.is_published)
print(workflow.updated_at)
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const workflow = await client.workflows.get("workflow_abc123xyz");

console.log(workflow.name);
console.log(workflow.is_published);
console.log(workflow.updated_at);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/workflow_abc123xyz' \
  -H 'accept: application/json' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
