---
title: 'Delete Workflow'
openapi: 'DELETE /v1/workflows/{workflow_id}'
---

Delete a workflow, its graph, and its run records.

This is irreversible. Workflow runs that are still pending or running are cancelled before deletion.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.workflows.delete("wf_abc123xyz")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

await client.workflows.delete("wf_abc123xyz");
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Workflows.Delete(ctx, "wf_abc123xyz"); err != nil {
		log.Fatal(err)
	}

	fmt.Println("deleted")
}
```

```bash curl
curl -X 'DELETE' \
  'https://api.retab.com/v1/workflows/wf_abc123xyz' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
