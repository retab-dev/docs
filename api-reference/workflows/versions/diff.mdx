---
title: "Diff workflow versions"
openapi: "GET /v1/workflows/versions/diff"
---

Compare two immutable workflow graph versions from the same workflow.

The response lists added, removed, and changed block and edge IDs. Fetch the
individual block or edge versions when you need field-level detail.

<RequestExample>

```python Python
diff = client.workflows.list_diff(
    workflow_id="wrk_abc123",
    from_workflow_version_id="ver_old",
    to_workflow_version_id="ver_new",
)
```

```typescript TypeScript
const diff = await client.workflows.list_diff({
  workflowId: "wrk_abc123",
  fromWorkflowVersionId: "ver_old",
  toWorkflowVersionId: "ver_new",
});
```

```go Go
diff, err := client.Workflows.ListDiff(ctx, &retab.WorkflowsListDiffParams{
	WorkflowID: "wrk_abc123", FromWorkflowVersionID: "ver_old", ToWorkflowVersionID: "ver_new",
})
```

```java Java
var diff = client.workflows().listDiff(
    WorkflowsListDiffParams.builder().workflowId("wrk_abc123").fromWorkflowVersionId("ver_old").toWorkflowVersionId("ver_new").build()
);
```

```bash cURL
curl -G https://api.retab.com/v1/workflows/versions/diff \
  -H "Api-Key: $RETAB_API_KEY" \
  --data-urlencode "workflow_id=wrk_abc123" \
  --data-urlencode "from_workflow_version_id=ver_old" \
  --data-urlencode "to_workflow_version_id=ver_new"
```

</RequestExample>
