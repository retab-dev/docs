---
title: "Get Workflow Version"
openapi: "GET /v1/workflows/versions/{workflow_version_id}"
---

Fetch one immutable workflow version.

Pass `workflow_id` with the request to disambiguate identical executable specs
that may appear in more than one workflow.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

version = client.workflows.get_version(
    "ver_abcdefghijklmnopqrstuvwxyz012345",
    workflow_id="wrk_abc123",
)
print(version.block_version_ids)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const version = await client.workflows.get_version(
  "ver_abcdefghijklmnopqrstuvwxyz012345",
  { workflowId: "wrk_abc123" },
);
console.log(version.blockVersionIds);
```

```go Go
version, err := client.Workflows.GetVersion(
	ctx,
	"ver_abcdefghijklmnopqrstuvwxyz012345",
	&retab.WorkflowsGetVersionParams{WorkflowID: "wrk_abc123"},
)
```

```java Java
var version = client.workflows().getVersion(
    "ver_abcdefghijklmnopqrstuvwxyz012345",
    WorkflowsGetVersionParams.builder().workflowId("wrk_abc123").build()
);
```

```bash cURL
curl -G https://api.retab.com/v1/workflows/versions/ver_abcdefghijklmnopqrstuvwxyz012345 \
  -H "Api-Key: $RETAB_API_KEY" \
  --data-urlencode "workflow_id=wrk_abc123"
```
