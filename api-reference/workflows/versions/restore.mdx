---
title: "Restore workflow version"
openapi: "POST /v1/workflows/versions/{workflow_version_id}/restore"
---

Restore an immutable workflow graph version into the workflow's current draft.

This creates a fresh draft graph from the selected version. The immutable
version row is not modified.

<RequestExample>

```python Python
workflow = client.workflows.create_version_restore("ver_abc123", workflow_id="wrk_abc123")
```

```typescript TypeScript
const workflow = await client.workflows.create_version_restore("ver_abc123", {
  workflowId: "wrk_abc123",
});
```

```go Go
workflow, err := client.Workflows.CreateVersionRestore(ctx, "ver_abc123", &retab.WorkflowsCreateVersionRestoreParams{
	WorkflowID: "wrk_abc123",
})
```

```java Java
var workflow = client.workflows().createVersionRestore(
    "ver_abc123",
    WorkflowsCreateVersionRestoreParams.builder().workflowId("wrk_abc123").build()
);
```

```bash cURL
curl -X POST "https://api.retab.com/v1/workflows/versions/ver_abc123/restore?workflow_id=wrk_abc123" \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>
