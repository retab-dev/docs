---
title: 'Get Block Test Run'
openapi: 'GET /v1/workflows/{workflow_id}/block-tests/{test_id}/runs/{run_id}'
---

Fetch a single immutable run record. Use this after polling
[Execute Block Tests](../execute) to inspect the inputs the block ran with,
the per-handle outputs, the assertion result, and any warnings or routing
decisions.

<RequestExample>

```python python
from retab import Retab

client = Retab()

run = client.workflows.tests.runs.get(
    workflow_id="wf_abc123xyz",
    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
    run_id="wfnodetestrun_tRNzPiMpBas4kDwD74M8d",
)

print(run.status)
print(run.outputs)              # ← per-handle outputs (renamed from `handle_outputs`)
print(run.assertion_result)
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const run = await client.workflows.tests.runs.get({
  workflowId: "wf_abc123xyz",
  testId: "wfnodetest_hsLEQiM61ez9Piv147MWk",
  runId: "wfnodetestrun_tRNzPiMpBas4kDwD74M8d",
});

console.log(run.status);
console.log(run.outputs);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/wf_abc123xyz/block-tests/wfnodetest_hsLEQiM61ez9Piv147MWk/runs/wfnodetestrun_tRNzPiMpBas4kDwD74M8d' \
  -H 'accept: application/json' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
