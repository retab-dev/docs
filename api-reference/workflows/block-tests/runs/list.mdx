---
title: 'List Block Test Runs'
openapi: 'GET /v1/workflows/{workflow_id}/block-tests/{test_id}/runs'
---

List the most recent run records for a block test, newest first. Each entry is
a full `WorkflowTestRunRecord` (target, source, fingerprints, outputs,
assertion result, verdict summary).

`limit` defaults to 20, max 100.

<RequestExample>

```python python
from retab import Retab

client = Retab()

runs = client.workflows.tests.runs.list(
    workflow_id="wf_abc123xyz",
    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
    limit=10,
)
for run in runs.runs:
    print(run.id, run.status, run.duration_ms)
```

```javascript javascript
import { Retab } from '/node';
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	runs, err := client.Workflows.Tests.Runs.List(ctx, "wf_abc123xyz", "wfnodetest_hsLEQiM61ez9Piv147MWk", 10)
	if err != nil {
		log.Fatal(err)
	}

	for _, run := range runs.Data {
		fmt.Println(run["id"], run["status"], run["duration_ms"])
	}
}
```
