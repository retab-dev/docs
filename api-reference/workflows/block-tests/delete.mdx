---
title: 'Delete Block Test'
openapi: 'DELETE /v1/workflows/{workflow_id}/block-tests/{test_id}'
---

Delete a block test. Returns 204 on success. Run records associated with the
deleted test are kept in the run-records collection (they remain queryable by
`run_record_id` if you cached one) but no longer surface from the test's `runs`
endpoint.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.workflows.tests.delete(
    workflow_id="wf_abc123xyz",
    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
)
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

await client.workflows.tests.delete({
  workflowId: "wf_abc123xyz",
  testId: "wfnodetest_hsLEQiM61ez9Piv147MWk",
});
```

```bash curl
curl -X 'DELETE' \
  'https://api.retab.com/v1/workflows/wf_abc123xyz/block-tests/wfnodetest_hsLEQiM61ez9Piv147MWk' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```http 204
HTTP/1.1 204 No Content
```

```json 404
{
  "detail": "Block test not found: wfnodetest_hsLEQiM61ez9Piv147MWk"
}
```

</ResponseExample>
