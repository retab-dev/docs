---
title: 'Get Run'
openapi: 'GET /v1/workflows/runs/{run_id}'
---

Get a single workflow run by ID. Use this endpoint to check the status of a running workflow or retrieve the results of a completed workflow.

<RequestExample>

```python python
from retab import Retab

client = Retab()

run = client.workflows.get_run("run_abc123xyz")

print(f"Status: {run.status}")
if run.status == "completed":
    print(f"Duration: {run.duration_ms}ms")
    print(f"Outputs: {run.final_outputs}")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const run = await client.workflows.getRun("run_abc123xyz");

console.log(`Status: ${run.status}`);
if (run.status === "completed") {
    console.log(`Duration: ${run.duration_ms}ms`);
    console.log("Outputs:", run.final_outputs);
}
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/runs/run_abc123xyz' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
