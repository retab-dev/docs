---
title: 'Update Workflow'
openapi: 'PATCH /v1/workflows/{workflow_id}'
---

Update workflow metadata. Only the fields you supply are touched — omit a field to leave it unchanged.

The email trigger allowlist is split into two arrays:

- `email_senders_whitelist` — full email addresses (e.g. `"ap@acme.com"`)
- `email_domains_whitelist` — bare domains (e.g. `"acme.com"`)

If a workflow has neither list configured, inbound email triggers are accepted from any sender.

<RequestExample>

```python python
from retab import Retab

client = Retab()

# Rename + update description
workflow = client.workflows.update(
    "wf_abc123xyz",
    name="Invoice Processing v2",
    description="Extract invoice fields, validate, and route for review",
)

# Configure email-trigger allowlist
workflow = client.workflows.update(
    "wf_abc123xyz",
    email_senders_whitelist=["ap@acme.com"],
    email_domains_whitelist=["acme.com", "acme-subsidiary.com"],
)
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	name := "Invoice Processing v2"
	description := "Extract invoice fields, validate, and route for review"
	workflow, err := client.Workflows.Update(ctx, "wf_abc123xyz", retab.UpdateWorkflowRequest{
		Name:        &name,
		Description: &description,
	})
	if err != nil {
		log.Fatal(err)
	}

	workflow, err = client.Workflows.Update(ctx, "wf_abc123xyz", retab.UpdateWorkflowRequest{
		EmailSendersWhitelist: []string{"ap@acme.com"},
		EmailDomainsWhitelist: []string{"acme.com", "acme-subsidiary.com"},
	})
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(workflow.Name)
}
```

```bash curl
curl -X 'PATCH' \

  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "name": "Invoice Processing v2",
    "description": "Extract invoice fields, validate, and route for review",
    "email_trigger": {
      "allowed_senders": ["ap@acme.com"],
      "allowed_domains": ["acme.com", "acme-subsidiary.com"]
    }
  }'
```
