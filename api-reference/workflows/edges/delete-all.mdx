---
title: "Delete All Edges"
openapi: "DELETE /v1/workflows/{workflow_id}/edges"
---

Delete every edge in a workflow's draft. Blocks are not touched.

This is useful when you want to rewire the entire graph without deleting and recreating the blocks themselves.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.workflows.edges.delete_all("wf_abc123xyz")
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

await client.workflows.edges.deleteAll("wf_abc123xyz");
```

```go go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Workflows.Edges.DeleteAll(ctx, "wf_abc123xyz"); err != nil {
		log.Fatal(err)
	}
}
```

```bash curl
curl -X 'DELETE' \
  'https://api.retab.com/v1/workflows/wf_abc123xyz/edges' \
  -H 'Api-Key: <your-api-key>'
```
