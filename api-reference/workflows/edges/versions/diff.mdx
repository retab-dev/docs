---
title: "Diff edge versions"
openapi: "GET /v1/workflows/edges/versions/diff"
---

Compare two immutable edge versions.

The response contains field-level changes between the two complete edge
snapshots.

<RequestExample>

```python Python
diff = client.workflows.edges.list_diff("ever_old", "ever_new")
```

```typescript TypeScript
const diff = await client.workflows.edges.list_diff({
  fromEdgeVersionId: "ever_old",
  toEdgeVersionId: "ever_new",
});
```

```go Go
diff, err := client.WorkflowEdges.ListDiff(ctx, &retab.WorkflowEdgesListDiffParams{
	FromEdgeVersionID: "ever_old", ToEdgeVersionID: "ever_new",
})
```

```java Java
var diff = client.workflowEdges().listDiff(
    WorkflowEdgesListDiffParams.builder().fromEdgeVersionId("ever_old").toEdgeVersionId("ever_new").build()
);
```

```bash cURL
curl -G https://api.retab.com/v1/workflows/edges/versions/diff \
  -H "Api-Key: $RETAB_API_KEY" \
  --data-urlencode "from_edge_version_id=ever_old" \
  --data-urlencode "to_edge_version_id=ever_new"
```

</RequestExample>
