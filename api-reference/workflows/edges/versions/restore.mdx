---
title: "Restore edge version"
openapi: "POST /v1/workflows/edges/versions/{edge_version_id}/restore"
---

Restore an immutable edge version into the workflow's current draft.

If the logical edge already exists in the current draft, its snapshot fields are
overwritten. If it does not exist, it is recreated.

<RequestExample>

```python Python
edge = client.workflows.edges.create_version_restore("ever_abc123")
```

```typescript TypeScript
const edge = await client.workflows.edges.create_version_restore("ever_abc123");
```

```go Go
edge, err := client.WorkflowEdges.CreateVersionRestore(ctx, "ever_abc123")
```

```java Java
var edge = client.workflowEdges().createVersionRestore("ever_abc123");
```

```bash cURL
curl -X POST https://api.retab.com/v1/workflows/edges/versions/ever_abc123/restore \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>
