---
title: "Restore Edge Version"
openapi: "POST /v1/workflows/edges/versions/{edge_version_id}/restore"
---

Restore a workflow edge version into the current workflow draft.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

edge = client.workflows.edges.create_version_restore("egv_abc123")
print(edge)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const edge = await client.workflows.edges.create_version_restore("egv_abc123");
console.log(edge);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	edge, err := client.Workflows.Edges.CreateVersionRestore(ctx, "egv_abc123")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(*edge)
}
```
