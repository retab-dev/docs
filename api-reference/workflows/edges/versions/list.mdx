---
title: "List Edge Versions"
openapi: "GET /v1/workflows/edges/versions"
---

List immutable edge versions for one workflow.

Use `edge_id` to narrow the list to one logical edge, or
`workflow_version_id` to inspect the edges that belong to one workflow version.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

versions = client.workflows.edges.list_versions(
    workflow_id="wrk_abc123",
    edge_id="edg_1",
)
print(versions.data[0].source)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const versions = await client.workflows.edges.list_versions({
  workflowId: "wrk_abc123",
  edgeId: "edg_1",
});
console.log(versions.data[0].source);
```

```go Go
versions, err := client.WorkflowEdges.ListVersions(ctx, &retab.WorkflowEdgesListVersionsParams{
	WorkflowID: "wrk_abc123",
	EdgeID: ptr("edg_1"),
})
```

```java Java
var versions = client.workflowEdges().listVersions(
    WorkflowEdgesListVersionsParams.builder()
        .workflowId("wrk_abc123")
        .edgeId("edg_1")
        .build()

```

