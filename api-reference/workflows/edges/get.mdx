---
title: "Get Edge"
openapi: "GET /v1/workflows/edges/{edge_id}?workflow_id={workflow_id}"
---

Get a single edge by ID.

<RequestExample>

```python python
from retab import Retab

client = Retab()

edge = client.workflows.edges.get(
    workflow_id="wf_abc123xyz",
    edge_id="edge-1",
)

print(f"{edge.source_block}/{edge.source_handle} -> {edge.target_block}/{edge.target_handle}")
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const edge = await client.workflows.edges.get("wf_abc123xyz", "edge-1");

console.log(
  `${edge.source_block}/${edge.source_handle} -> ${edge.target_block}/${edge.target_handle}`,
);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	edge, err := client.Workflows.Edges.Get(ctx, "wf_abc123xyz", "edge-1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Printf("%s/%s -> %s/%s\n", edge.SourceBlock, edge.SourceHandle, edge.TargetBlock, edge.TargetHandle)
}
```

```bash curl
curl -X 'GET' \

```
