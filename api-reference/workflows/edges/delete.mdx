---
title: "Delete Edge"
openapi: "DELETE /v1/workflows/edges/{edge_id}"
---

Delete a single edge.

To remove an entire block and all its incident edges in one call, use [Delete Block](/api-reference/workflows/blocks/delete) — it cleans up edges automatically.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

client.workflows.edges.delete(
    "edge-extract-to-review",
)
```
