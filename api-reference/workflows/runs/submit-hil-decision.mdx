---
title: 'Submit HIL Decision'
openapi: 'POST /v1/workflows/runs/{run_id}/hil-decisions'
---

Submit an approve/reject decision for a human-in-the-loop block on a workflow run.

This endpoint is command-oriented:
- it records the decision with Temporal as the source of truth
- it returns decision receipt state, not a refreshed `WorkflowRun`
- the run projection in Mongo may update shortly afterward

Use [Get Run](/api-reference/workflows/runs/get) to refresh the projected run state after submission, or [Get HIL Decision](/api-reference/workflows/runs/get-hil-decision) when you need command-side decision truth.

Common outcomes:
- `submission_status="accepted"`: a new matching decision was accepted
- `submission_status="already_received"`: the same logical decision had already been received

If the block already has the opposite decision or is no longer eligible to accept a decision, the endpoint returns `409`.

<RequestExample>

```python python
from retab import Retab

client = Retab()

# Approve a HIL block, optionally with edited data
result = client.workflows.runs.submit_hil_decision(
    run_id="run_abc123",
    block_id="hil-1",
    approved=True,
    modified_data={"amount": 1234.56},
    command_id="cmd_idempotent_abc",  # optional dedup key
)

print(result.submission_status)  # "accepted" / "already_received"

# Reject a HIL block
result = client.workflows.runs.submit_hil_decision(
    run_id="run_abc123",
    block_id="hil-1",
    approved=False,
)
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

// Approve a HIL block, optionally with edited data
const result = await client.workflows.runs.submitHilDecision("run_abc123", {
  blockId: "hil-1",
  approved: true,
  modifiedData: { amount: 1234.56 },
  commandId: "cmd_idempotent_abc",  // optional dedup key
});

console.log(result.submission_status);

// Reject a HIL block
await client.workflows.runs.submitHilDecision("run_abc123", {
  blockId: "hil-1",
  approved: false,
});
```

```bash curl
# Approve with edited data
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/runs/run_abc123/hil-decisions' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "block_id": "hil-1",
    "approved": true,
    "modified_data": { "amount": 1234.56 },
    "command_id": "cmd_idempotent_abc"
  }'
```
