---
title: 'Get Artifact'
openapi: 'GET /v1/workflows/runs/{run_id}/artifacts/{operation}/{artifact_id}'
---

Get the workflow-native artifact payload referenced by a workflow run step.

Use this endpoint when a step returns an artifact reference whose `operation` does not map to a typed resource API such as extractions, classifications, splits, parses, edits, or partitions.

<RequestExample>

```python python
import requests

response = requests.get(
    "https://api.retab.com/v1/workflows/runs/run_abc123xyz/artifacts/function/fn_artifact_abc123",
    headers={"Api-Key": "<your-api-key>"},
    timeout=30,
)
response.raise_for_status()

artifact = response.json()
print(artifact["data"])
```

```javascript javascript
const response = await fetch(
  "https://api.retab.com/v1/workflows/runs/run_abc123xyz/artifacts/function/fn_artifact_abc123",
  {
    headers: { "Api-Key": "<your-api-key>" },
  },
);

const artifact = await response.json();
console.log(artifact.data);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/runs/run_abc123xyz/artifacts/function/fn_artifact_abc123' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
