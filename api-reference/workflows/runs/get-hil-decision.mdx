---
title: 'Get HIL Decision'
openapi: 'GET /v1/workflows/runs/{run_id}/hil-decisions/{block_id}'
---

Get Temporal-owned decision state for a single human-in-the-loop block in a workflow run.

This endpoint is useful when:
- you want to confirm whether a decision was received
- you need to distinguish `decision_received` from `decision_applied`
- the run projection may still be catching up after a recent submission

If the block exists but no decision has been submitted yet, the response still returns `200` with:
- `decision_received=false`
- `decision_applied=false`
- `approved=null`

Use [Submit HIL Decision](/api-reference/workflows/runs/submit-hil-decision) to submit the decision, and [Get Run](/api-reference/workflows/runs/get) to inspect the projected run state.

<RequestExample>

```python python
from retab import Retab

client = Retab()

decision = client.workflows.runs.get_hil_decision(
    run_id="run_abc123",
    block_id="hil-1",
)

print(decision.decision_received, decision.decision_applied, decision.approved)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const decision = await client.workflows.runs.getHilDecision("run_abc123", "hil-1");

console.log(decision.decision_received, decision.decision_applied, decision.approved);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	decision, err := client.Workflows.Runs.GetHILDecision(ctx, "run_abc123", "hil-1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(decision.DecisionReceived, decision.DecisionApplied, decision.Approved)
}
```
