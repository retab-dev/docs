---
title: "Delete Run"
openapi: "DELETE /v1/workflows/runs/{run_id}"
---

Delete a workflow run and its persisted step data. This is irreversible.

If the run is still active, prefer [Cancel Run](/api-reference/workflows/runs/cancel) — that records a clean `cancelled` outcome instead of removing the record entirely.

<RequestExample>

```python python
from retab import Retab

client = Retab()

client.workflows.runs.delete("run_abc123")
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

await client.workflows.runs.delete("run_abc123");
```

```go go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Workflows.Runs.Delete(ctx, "run_abc123"); err != nil {
		log.Fatal(err)
	}
}
```

```bash curl
curl -X 'DELETE' \
  'https://api.retab.com/v1/workflows/runs/run_abc123' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
