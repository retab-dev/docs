---
title: "Delete Run"
openapi: "DELETE /v1/workflows/runs/{run_id}"
---

Delete a workflow run and its persisted step data. This is irreversible.

If the run is still active, prefer [Cancel Run](/api-reference/workflows/runs/cancel) — that records a clean `cancelled` outcome instead of removing the record entirely.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

client.workflows.runs.delete("run_abc123")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

await client.workflows.runs.delete("run_abc123");
```

}
