---
title: 'Get Step'
openapi: 'GET /v1/workflows/runs/{run_id}/steps/{node_id}'
---

Get step status and handle data for a specific step in a workflow run. Use this to inspect the normalized inputs and outputs for a particular node after or during execution.

<RequestExample>

```python python
from retab import Retab

client = Retab()

step = client.workflows.runs.steps.get("run_abc123xyz", "extract-node-1")

print(step.status)
print(step.extracted_data)

if step.handle_outputs:
    payload = step.handle_outputs["output-json-0"]
    print(payload.type)
    print(payload.data)
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const step = await client.workflows.runs.steps.get("run_abc123xyz", "extract-node-1");

console.log(`Step status: ${step.status}`);
console.log("Handles:", step.handle_outputs);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/runs/run_abc123xyz/steps/extract-node-1' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
