---
title: 'Get Step'
openapi: 'GET /v1/workflows/runs/{run_id}/steps/{block_id}'
---

Get step lifecycle and handle data for a specific step in a workflow run. Use this to inspect the normalized inputs and outputs for a particular block after or during execution.

<RequestExample>

```python python
from retab import Retab

client = Retab()

step = client.workflows.runs.steps.get("run_abc123xyz", "extract-block-1")

print(step.lifecycle.status)
print(step.extracted_data)

# Dereference the artifact record produced by this step:
if step.artifact:
    artifact = client.workflows.artifacts.get(step.artifact)
    print(artifact.operation)
    print(artifact.id)

# Or inspect raw handle payloads:
if step.handle_outputs:
    payload = step.handle_outputs["output-json-0"]
    print(payload.type)
    print(payload.data)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const step = await client.workflows.runs.steps.get("run_abc123xyz", "extract-block-1");

console.log(`Step lifecycle: ${step.lifecycle.status}`);
const payload = step.handle_outputs["output-json-0"];
if (payload?.type === "json") {
    console.log(payload.data);
}
if (step.artifact) {
    const artifact = await client.workflows.artifacts.get(step.artifact);
    console.log(artifact);
}
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	step, err := client.Workflows.Runs.Steps.Get(ctx, "run_abc123xyz", "extract-block-1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Printf("Step lifecycle: %s\n", step.Lifecycle["status"])
	if step.Artifact != nil {
		fmt.Println(step.Artifact.Operation, step.Artifact.ID)
	}

	// Or inspect raw handle payloads:
	if payload, ok := step.HandleOutputs["output-json-0"]; ok {
		fmt.Println(payload.Type)
		fmt.Println(payload.Data)
	}
}
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/runs/run_abc123xyz/steps/extract-block-1' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>

`artifact` is a pointer, not the artifact payload. Use
[Get Artifact](/api-reference/workflows/artifacts/get) when you need the full
record for a persisted step result, such as HIL condition evaluations,
function output, or API call retry attempts.
