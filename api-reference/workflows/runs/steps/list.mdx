---
title: 'Get Steps (Batch)'
openapi: 'POST /v1/workflows/runs/{run_id}/steps/batch'
---

Get outputs for multiple steps in a single request. This is useful when you need to retrieve data from several nodes at once without making individual requests for each.

<RequestExample>

```python python
from retab import Retab

client = Retab()

batch = client.workflows.runs.steps.get_batch(
    "run_abc123xyz",
    ["extract-node-1", "extract-node-2", "split-node-1"]
)

for node_id, step in batch.outputs.items():
    print(f"{node_id}: {step.status}")
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const batch = await client.workflows.runs.steps.getBatch(
    "run_abc123xyz",
    ["extract-node-1", "extract-node-2", "split-node-1"]
);

for (const [nodeId, step] of Object.entries(batch.outputs)) {
    console.log(`${nodeId}: ${step.status}`);
}
```

```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/runs/run_abc123xyz/steps/batch' \
  -H 'Content-Type: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -d '{
    "node_ids": ["extract-node-1", "extract-node-2", "split-node-1"]
  }'
```
