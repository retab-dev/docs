---
title: 'List Steps'
openapi: 'GET /v1/workflows/runs/{run_id}/steps'
---

Retrieve the persisted step list for a workflow run. Use this when you want the complete set of step documents for a run, including lifecycle states, handle inputs/outputs, and loop metadata.

Steps include an `artifact` ref when the block produced a persisted record.
Use [Get Artifact](/api-reference/workflows/artifacts/get) to dereference one
ref, or [List Artifacts](/api-reference/workflows/artifacts/list) to fetch all
artifact records for the run.

<RequestExample>

```python python
from retab import Retab

client = Retab()

steps = client.workflows.runs.steps.list("run_abc123xyz")

for step in steps:
    print(step.block_id, step.lifecycle.status, step.artifact)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const steps = await client.workflows.runs.steps.list("run_abc123xyz");

for (const step of steps) {
    console.log(step.block_id, step.lifecycle.status, step.artifact);
}
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	steps, err := client.Workflows.Runs.Steps.List(ctx, "run_abc123xyz")
	if err != nil {
		log.Fatal(err)
	}

	for _, step := range steps {
		fmt.Println(step.BlockID, step.Lifecycle["status"])
	}
}
```
