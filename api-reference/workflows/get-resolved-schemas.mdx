---
title: 'Get Workflow Resolved Schemas'
openapi: 'GET /v1/workflows/{workflow_id}/resolved-schemas'
---

Get graph-derived input/output schemas for every block in the workflow's current draft.

<RequestExample>

```python python
from retab import Retab

client = Retab()

schemas = client.workflows.get_resolved_schemas("wf_abc123xyz")
print(schemas.schemas["extract-1"].output_schemas)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();

const schemas = await client.workflows.getResolvedSchemas("wf_abc123xyz");
console.log(schemas.schemas["extract-1"].output_schemas);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	schemas, err := client.Workflows.GetResolvedSchemas(ctx, "wf_abc123xyz")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(schemas.Schemas["extract-1"].OutputSchemas)
}
```

```bash curl
curl -X 'GET' \
```
