---
title: "Create Workflow Test Run"
openapi: "POST /v1/workflows/tests/runs"
---

Create a workflow-test run against the current workflow draft. A run can execute
one saved test, every test for one block, or every test in the workflow.

The canonical route is flat: send `workflow_id` in the request body when
running a workflow-scoped target or all tests. When `test_id` is supplied, the
workflow can be derived from the saved test.

The response is a run resource. Use its `id` with the run-id-first endpoints:
[Get Workflow Test Run](/api-reference/workflows/tests/runs/get), [List Test
Run Results](/api-reference/workflows/tests/results/list).

The request body is one of three shapes:

- **`test_id`** - run a single test by id; `workflow_id` may be omitted.
- **`workflow_id` + `target`** - run every test for one block (`{ "type": "block", "block_id": "..." }`).
- **`workflow_id` only** - run every test in the workflow.

`n_consensus` is optional and only meaningful for extract, split, and classifier
blocks. Allowed values are `3`, `5`, or `7`; the override applies to this run
only.

```python Python
from retab import Retab

client = Retab()

run = client.workflows.tests.runs.create(

    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
)

print(run.id, run.lifecycle.status)
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	run, err := client.Workflows.Tests.Runs.Create(ctx, retab.CreateWorkflowTestRunRequest{
		WorkflowID: "wf_abc123xyz",
		TestID:     "wfnodetest_hsLEQiM61ez9Piv147MWk",
	})
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(run.ID, run.Lifecycle.Status)
}
```

```rust Rust
use retab::models::CreateWorkflowTestRunForTestRequest;
use retab::resources::workflow_test_runs::CreateParams;
use retab::Retab;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Retab::new(std::env::var("RETAB_API_KEY")?);

    let mut for_test = CreateWorkflowTestRunForTestRequest::new(
        "wfnodetest_hsLEQiM61ez9Piv147MWk",
    );
    for_test.workflow_id = Some("wf_abc123xyz".into());

    let run = client
        .workflow_test_runs()
        .create(CreateParams::new(for_test.into()))
        .await?;
    println!("{} {:?}", run.id, run.lifecycle);
    Ok(())
}
```
