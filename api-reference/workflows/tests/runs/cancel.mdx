---
title: "Cancel Workflow Test Run"
openapi: "POST /v1/workflows/tests/runs/{run_id}/cancel"
---

Cancel a pending or running workflow-test run by `run_id`. Cancellation is
run-specific; you do not pass the workflow id or test id when cancelling.

<RequestExample>

```python python
from retab import Retab

client = Retab()

run = client.workflows.tests.runs.cancel("wftestrun_q1z2")
print(run.lifecycle.status)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const run = await client.workflows.tests.runs.cancel("wftestrun_q1z2");
console.log(run.lifecycle.status);
```

```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/workflows/tests/runs/wftestrun_q1z2/cancel' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>

<ResponseExample>
