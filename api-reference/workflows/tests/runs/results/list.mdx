---
title: "List Test Run Results"
openapi: "GET /v1/workflows/tests/runs/{run_id}/results"
---

List the test results produced by one workflow-test run.

The response uses the canonical Retab list envelope:
`{ "data": [...], "list_metadata": { "before": null, "after": null } }`.

<RequestExample>

```python python
from retab import Retab

client = Retab()

results = client.workflows.tests.runs.results.list("wftestrun_q1z2")
for result in results.data:
    print(result.test_id, result.verdict.status)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const results = await client.workflows.tests.runs.results.list({
  runId: "wftestrun_q1z2",
});
for (const result of results.data) {
  console.log(result.test_id, result.verdict.status);
}
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	results, err := client.Workflows.Tests.Runs.Results.List(ctx, "wftestrun_q1z2")
	if err != nil {
		log.Fatal(err)
	}

	for _, result := range results.Data {
		fmt.Println(result["test_id"], result["verdict"])
	}
}
```
