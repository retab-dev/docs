---
title: "Get Test Run Result"
openapi: "GET /v1/workflows/tests/runs/{run_id}/results/{test_id}"
---

Fetch one result from a workflow-test run. The child key is the saved
`test_id`, not an internal result row id.

<RequestExample>

```python python
from retab import Retab

client = Retab()

result = client.workflows.tests.runs.results.get(
    "wftestrun_q1z2",
    "wfnodetest_a",
)
print(result.verdict.status, result.outputs)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const result = await client.workflows.tests.runs.results.get(
  "wftestrun_q1z2",
  "wfnodetest_a",
);
console.log(result.verdict.status, result.outputs);
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/tests/runs/wftestrun_q1z2/results/wfnodetest_a' \
  -H 'Api-Key: <your-api-key>'
```

</RequestExample>
