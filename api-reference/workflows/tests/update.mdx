---
title: "Update Workflow Test"
openapi: "PATCH /v1/workflows/tests/{test_id}"
---

Patch the name, assertion, and/or source of a workflow test. Every field is
optional in the request body — fields you omit are left untouched. Setting
`assertion: null` is rejected explicitly (use [Delete Workflow Test](./delete) if
you want to remove the test entirely).

Updating `source` rebinds where the inputs come from:

- `{ type: "manual", handle_inputs: ... }` — switch to hand-written inputs.
- `{ type: "run_step", run_id: ..., step_id: ... }` — re-capture inputs from
  a different workflow run / step.

The response is the full updated `WorkflowTest`.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

# Rename only.
test = client.workflows.tests.update(
    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
    name="Q1 invoice — vendor name check",
)

# Replace the assertion.
test = client.workflows.tests.update(
    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
    assertion={
        "target": {"output_handle_id": "output-json-0", "path": "vendor.name"},
        "condition": {"kind": "matches_regex", "expected": r"^Acme.*"},
    },
)

# Re-capture inputs from a different run.
test = client.workflows.tests.update(
    test_id="wfnodetest_hsLEQiM61ez9Piv147MWk",
    source={
        "type": "run_step",
        "run_id": "wfrun_new789",
        "step_id": "block_extract_invoice",
    },
)
```

```go Go
			"type":    "run_step",
```

```rust Rust
use retab::models::{RunStepWorkflowTestSource, UpdateWorkflowTestRequest};
use retab::resources::workflow_tests::UpdateParams;
use retab::Retab;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Retab::new(std::env::var("RETAB_API_KEY")?);
    let test_id = "wfnodetest_hsLEQiM61ez9Piv147MWk";

    // Rename only.
    let renamed_body = UpdateWorkflowTestRequest {
        name: Some("Q1 invoice — vendor name check".into()),
        ..Default::default()
    };
    let _renamed = client
        .workflow_tests()
        .update(test_id, UpdateParams::new(renamed_body))
        .await?;

    // Re-capture inputs from a different run.
    let mut source = RunStepWorkflowTestSource::new("wfrun_new789");
    source.step_id = Some("block_extract_invoice".into());
    let resource_body = UpdateWorkflowTestRequest {
        source: Some(source.into()),
        ..Default::default()
    };
    let _rebound = client
        .workflow_tests()
        .update(test_id, UpdateParams::new(resource_body))
        .await?;
    Ok(())
}
```
