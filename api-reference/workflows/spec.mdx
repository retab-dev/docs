---
title: "Export Workflow Spec"
openapi: "GET /v1/workflows/{workflow_id}/spec"
---

Export the current draft workflow as canonical declarative YAML.

Exports use `apiVersion: workflows.retab.com/v1alpha2` and include raw explicit handles on every non-synthetic edge.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

exported = client.workflows.spec.get("wf_abc123")

print(exported.yaml_definition)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const exported = await client.workflows.specs.get("wf_abc123");

console.log(exported.yamlDefinition);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	exported, err := client.Workflows.Spec.Export(ctx, "wf_abc123")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println((*exported)["yaml_definition"])
}
```
