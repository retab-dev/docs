---
title: "Get Step"
openapi: "GET /v1/workflows/steps/{step_id}"
---

Get one joined step row by `step_id`. The response matches a single row from
[Query Steps](/api-reference/workflows/steps/query), including status, handle
data, and fingerprint metadata.

<RequestExample>

```python python
from retab import Retab

client = Retab()

step = client.workflows.steps.get("step_extract_1")

print(step.status)
print(step.handle_outputs)

if step.handle_outputs:
    payload = step.handle_outputs["output-json-0"]
    print(payload.type)
    print(payload.data)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const step = await client.workflows.steps.get("step_extract_1");

console.log(`Step status: ${step.status}`);
const payload = step.handle_outputs["output-json-0"];
if (payload?.type === "json") {
  console.log(payload.data);
}
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	step, err := client.Workflows.Steps.Get(ctx, "step_extract_1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Printf("Step status: %s\n", step.Status)
	if payload, ok := step.HandleOutputs["output-json-0"]; ok {
		fmt.Println(payload.Type)
		fmt.Println(payload.Data)
	}
}
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/steps/step_extract_1' \
  -H 'Api-Key: <your-api-key>'
```
