---
title: "Delete Block"
openapi: "DELETE /v1/workflows/blocks/{block_id}"
---

Delete a block from a workflow's draft. Every edge connected to the block (incoming or outgoing) is deleted with it — you don't need to clean those up first.

If the block is a container (`while_loop`, `for_each`), its child blocks become orphaned (their `parent_id` is reset to `null`) but are otherwise preserved.

<RequestExample>

from retab import Retab

client = Retab()

client.workflows.blocks.delete(
    "extract-1",
)
