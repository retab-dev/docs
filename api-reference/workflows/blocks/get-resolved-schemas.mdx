---
title: 'Get Block Resolved Schemas'
openapi: 'GET /v1/workflows/{workflow_id}/blocks/{block_id}/resolved-schemas'
---

Get graph-derived input/output schemas for one block in the workflow's current draft.

<RequestExample>

```python python
from retab import Retab

client = Retab()

result = client.workflows.blocks.get_resolved_schemas(
    workflow_id="wf_abc123xyz",
    block_id="extract-1",
)

print(result.block_schema.output_schemas)
```

```javascript javascript
import { Retab } from '/node';

const client = new Retab();

const result = await client.workflows.blocks.getResolvedSchemas(
  "wf_abc123xyz",
  "extract-1",
);

console.log(result.schema.output_schemas);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	result, err := client.Workflows.Blocks.GetResolvedSchemas(ctx, "wf_abc123xyz", "extract-1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(result.Schema.OutputSchemas)
}
```

