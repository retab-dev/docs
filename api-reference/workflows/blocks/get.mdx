---
title: "Get Block"
openapi: "GET /v1/workflows/blocks/{block_id}?workflow_id={workflow_id}"
---

Get a single authored block from the workflow's current draft.

<RequestExample>

```python python
from retab import Retab

client = Retab()

block = client.workflows.blocks.get(
    workflow_id="wf_abc123xyz",
    block_id="extract-1",
)

print(block.type, block.label)
print(block.config)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const block = await client.workflows.blocks.get("wf_abc123xyz", "extract-1");

console.log(block.type, block.label);
console.log(block.config);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	block, err := client.Workflows.Blocks.Get(ctx, "wf_abc123xyz", "extract-1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(block.Type, block.Label)
	fmt.Println(block.Config)
}
```

```bash curl
curl -X 'GET' \

  -H 'Api-Key: <your-api-key>'
```
