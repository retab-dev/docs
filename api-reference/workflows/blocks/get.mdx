---
title: 'Get Block'
openapi: 'GET /v1/workflows/{workflow_id}/blocks/{block_id}'
---

Get a single block from the workflow's current draft, including its `config` and `resolved_schemas`.

<RequestExample>

```python python
from retab import Retab

client = Retab()

block = client.workflows.blocks.get(
    workflow_id="wf_abc123xyz",
    block_id="extract-1",
)

print(block.type, block.label)
print(block.config)
print(block.resolved_schemas.output_schemas)
```

```javascript javascript
import { Retab } from '/node';

const client = new Retab();

const block = await client.workflows.blocks.get("wf_abc123xyz", "extract-1");

console.log(block.type, block.label);
console.log(block.config);
console.log(block.resolved_schemas?.output_schemas);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	block, err := client.Workflows.Blocks.Get(ctx, "wf_abc123xyz", "extract-1")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(block.Type, block.Label)
	fmt.Println(block.Config)
	if block.ResolvedSchemas != nil {
		fmt.Println(block.ResolvedSchemas.OutputSchemas)
	}
}
```

```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/workflows/wf_abc123xyz/blocks/extract-1' \
  -H 'Api-Key: <your-api-key>'
```
