---
title: "Get Block Version"
openapi: "GET /v1/workflows/blocks/versions/{block_version_id}"
---

Fetch one immutable block version by its `bver_...` id.

A block version is a full snapshot of the block inside one workflow version.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

version = client.workflows.blocks.get_version("bver_abcdefghijklmnopqrstuvwxyz012345")
print(version.config)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const version = await client.workflows.blocks.get_version(
  "bver_abcdefghijklmnopqrstuvwxyz012345",
);
console.log(version.config);
```

```go Go
version, err := client.WorkflowBlocks.GetVersion(
	ctx,
	"bver_abcdefghijklmnopqrstuvwxyz012345",
)
```

```java Java
var version = client.workflowBlocks().getVersion(
    "bver_abcdefghijklmnopqrstuvwxyz012345"
);
```

```bash cURL
curl https://api.retab.com/v1/workflows/blocks/versions/bver_abcdefghijklmnopqrstuvwxyz012345 \
  -H "Api-Key: $RETAB_API_KEY"
```
