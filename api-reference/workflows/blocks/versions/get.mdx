---
title: "Get Block Version"
openapi: "GET /v1/workflows/blocks/versions/{block_version_id}"
---

Fetch one immutable workflow block version.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

version = client.workflows.blocks.get_version("bkv_abc123")
print(version)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const version = await client.workflows.blocks.get_version("bkv_abc123");
console.log(version);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	version, err := client.Workflows.Blocks.GetVersion(ctx, "bkv_abc123")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(*version)
}
```
