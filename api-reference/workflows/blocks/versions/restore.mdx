---
title: "Restore block version"
openapi: "POST /v1/workflows/blocks/versions/{block_version_id}/restore"
---

Restore an immutable block version into the workflow's current draft.

If the logical block already exists in the current draft, its snapshot fields
are overwritten. If it does not exist, it is recreated.

<RequestExample>

```python Python
block = client.workflows.blocks.create_version_restore("bver_abc123")
```

```typescript TypeScript
const block = await client.workflows.blocks.create_version_restore("bver_abc123");
```

```go Go
block, err := client.WorkflowBlocks.CreateVersionRestore(ctx, "bver_abc123")
```

```java Java
var block = client.workflowBlocks().createVersionRestore("bver_abc123");
```

```bash cURL
curl -X POST https://api.retab.com/v1/workflows/blocks/versions/bver_abc123/restore \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>
