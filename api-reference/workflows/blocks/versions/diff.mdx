---
title: "Diff block versions"
openapi: "GET /v1/workflows/blocks/versions/diff"
---

Compare two immutable block versions.

The response contains field-level changes between the two complete block
snapshots.

<RequestExample>

```python Python
diff = client.workflows.blocks.list_diff("bver_old", "bver_new")
```

```typescript TypeScript
const diff = await client.workflows.blocks.list_diff({
  fromBlockVersionId: "bver_old",
  toBlockVersionId: "bver_new",
});
```

```go Go
diff, err := client.WorkflowBlocks.ListDiff(ctx, &retab.WorkflowBlocksListDiffParams{
	FromBlockVersionID: "bver_old", ToBlockVersionID: "bver_new",
})
```

```java Java
var diff = client.workflowBlocks().listDiff(
    WorkflowBlocksListDiffParams.builder().fromBlockVersionId("bver_old").toBlockVersionId("bver_new").build()
);
```

```bash cURL
curl -G https://api.retab.com/v1/workflows/blocks/versions/diff \
  -H "Api-Key: $RETAB_API_KEY" \
  --data-urlencode "from_block_version_id=bver_old" \
  --data-urlencode "to_block_version_id=bver_new"
```

</RequestExample>
