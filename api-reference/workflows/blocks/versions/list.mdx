---
title: "List Block Versions"
openapi: "GET /v1/workflows/blocks/versions"
---

List immutable block versions for one workflow.

Use `block_id` to narrow the list to one logical block, or
`workflow_version_id` to inspect the blocks that belong to one workflow version.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

versions = client.workflows.blocks.list_versions(
    workflow_id="wrk_abc123",
    block_id="blk_extract",
)
print(versions.data[0].config)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const versions = await client.workflows.blocks.list_versions({
  workflowId: "wrk_abc123",
  blockId: "blk_extract",
});
console.log(versions.data[0].config);
```

```go Go
versions, err := client.WorkflowBlocks.ListVersions(ctx, &retab.WorkflowBlocksListVersionsParams{
	WorkflowID: "wrk_abc123",
	BlockID: ptr("blk_extract"),
})
```

```java Java
var versions = client.workflowBlocks().listVersions(
    WorkflowBlocksListVersionsParams.builder()
        .workflowId("wrk_abc123")
        .blockId("blk_extract")
        .build()
);
```

```bash cURL
curl -G https://api.retab.com/v1/workflows/blocks/versions \
  -H "Api-Key: $RETAB_API_KEY" \
  --data-urlencode "workflow_id=wrk_abc123" \
  --data-urlencode "block_id=blk_extract"
```
