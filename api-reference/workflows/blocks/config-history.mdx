---
title: "List Block Config History"
openapi: "GET /v1/workflows/{workflow_id}/blocks/{block_id}/config-history"
---

Return the config-version timeline for a single block. Consecutive workflow
publishes in which the block's config did not change are grouped into a
single entry, with the configuration fingerprint, the captured config
snapshot, the range of snapshot versions the era spans, and the run count
recorded against that era.

Returns the canonical `{ "data": [...], "list_metadata": { "before": null, "after": null } }` pagination envelope shared with all Retab list endpoints. Cursor pagination is not yet implemented for this endpoint — `list_metadata` is always `{ before: null, after: null }`.

<RequestExample>

```python python
from retab import Retab

client = Retab()

history = client.workflows.blocks.config_history("wf_abc123xyz", "extract-1")

for entry in history.data:
    print(entry.config_fingerprint, entry.snapshot_versions, entry.run_count)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();

const history = await client.workflows.blocks.configHistory(

```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	history, err := client.Workflows.Blocks.ConfigHistory(ctx, "wf_abc123xyz", "extract-1")
	if err != nil {
		log.Fatal(err)
	}

	for _, entry := range history.Data {
		fmt.Println(entry.ConfigFingerprint, entry.SnapshotVersions, entry.RunCount)
	}
}
```
