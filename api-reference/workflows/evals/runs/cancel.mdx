---
title: "Cancel Workflow Eval Run"
openapi: "POST /v1/workflows/evals/runs/{run_id}/cancel"
---

Cancel a pending or running workflow-eval run by `run_id`. Cancellation is
run-specific; you do not pass the workflow id or eval id when cancelling.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

run = client.workflows.evals.runs.cancel("wfevalrun_q1z2")
print(run.lifecycle.status)
```
