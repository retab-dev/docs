---
title: "List Secrets"
openapi: "GET /v1/secrets"
---

List secret metadata for the current environment. Secret values are not returned.

```python Python
secrets = client.secrets.list_secrets()
```

```typescript TypeScript
const secrets = await client.secrets.list_secrets();
```

```go Go
secrets, err := client.Secrets.List(ctx)
```

```java Java
// List secrets with the Retab Java client.
```

```bash cURL
curl https://api.retab.com/v1/secrets -H "Authorization: Bearer $RETAB_API_KEY"
```
