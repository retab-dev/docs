---
title: "Delete Secret"
openapi: "DELETE /v1/secrets/{name}"
---

Delete an environment-scoped secret. Workflows that reference the deleted secret
will fail to resolve it at runtime.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

client.secrets.delete_secret("RESEND_API_KEY")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

await client.secrets.delete_secret("RESEND_API_KEY");
```

```go Go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Secrets.Delete(ctx, "RESEND_API_KEY"); err != nil {
		log.Fatal(err)
	}
}
```
