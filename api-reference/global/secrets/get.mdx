---
title: 'Get'
openapi: 'GET /v1/secrets/external_api_keys/{provider}'
---


<RequestExample>

```python python
from retab import Retab

reclient = Retab()
response = reclient.secrets.external_api_keys.get(
    provider = "OpenAI"
)
```


```bash curl
curl -X 'GET' \
  'https://api.retab.com/v1/secrets/external_api_keys/OpenAI' \
  -H 'accept: application/json' \
  -H 'Api-Key: RETAB_API_KEY'
```


</RequestExample>
