---
title: 'Delete'
openapi: 'DELETE /v1/secrets/external_api_keys/{provider}'
---

<RequestExample>

```python python
from retab.client import Retab

reclient = Retab()
response = reclient.secrets.external_api_keys.delete(
    provider = "OpenAI"
)
```


```bash curl
curl -X 'DELETE' \
  'https://api.retab.com/v1/secrets/external_api_keys/OpenAI' \
  -H 'accept: application/json' \
  -H 'Api-Key: RETAB_API_KEY'
```


</RequestExample>


<ResponseExample>

```json 200
{
  "status": "success",
  "message": "OpenAI API key deleted successfully"
}
```



</ResponseExample>
