---
title: 'Create'
openapi: 'POST /v1/secrets/external_api_keys'
---



<RequestExample>




```python python
from retab.client import Retab

reclient = Retab()
response = reclient.secrets.external_api_keys.create(
    provider = "OpenAI",
    api_key = "sk-..."
)
```


```bash curl
curl -X 'POST' \
  'https://api.retab.com/v1/secrets/external_api_keys' \
  -H 'accept: application/json' \
  -H 'Api-Key: RETAB_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
  "provider": "OpenAI",
  "api_key": "sk-..."
}'
```


</RequestExample>


<ResponseExample>

```json 200
{
  "status": "success",
  "message": "OpenAI API key updated successfully"
}
```

</ResponseExample>
