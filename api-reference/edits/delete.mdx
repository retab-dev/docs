---
title: 'Delete Edit'
openapi: 'DELETE /v1/edits/{edit_id}'
---

Delete a persisted `Edit` record by ID. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()
client.edits.delete("edit_01G34H8J2K")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();
await client.edits.delete("edit_01G34H8J2K");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/edits/edit_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```text 204
No Content
```

```json 404
{
  "detail": "Edit edit_01G34H8J2K not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="edit_id" type="string" required>
  The unique identifier of the edit to delete.
</ParamField>
