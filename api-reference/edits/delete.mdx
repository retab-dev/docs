---
title: "Delete Edit"
openapi: "DELETE /v1/edits/{edit_id}"
---

Delete a persisted `Edit` record by ID. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()
client.edits.delete("edit_01G34H8J2K")
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });
await client.edits.delete("edit_01G34H8J2K");
```

```go go
package main

import (
	"context"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	if err := client.Edits.Delete(ctx, "edit_01G34H8J2K"); err != nil {
		log.Fatal(err)
	}
}
```

