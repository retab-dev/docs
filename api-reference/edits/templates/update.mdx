---
title: "Update Template"
openapi: "PATCH /v1/edits/templates/{template_id}"
---

Update the name and/or the form fields of an existing template. Both fields are optional; only supplied values are overwritten. Supplying `form_fields` replaces the entire list.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

template = client.edits.templates.update(
    "edittplt_abc123",
    name="W-9 Form (2024)",
)
print(template.name)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });
const template = await client.edits.templates.update("edittplt_abc123", "W-9 Form (2024)");

```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	name := "W-9 Form (2024)"
	template, err := client.Edits.Templates.Update(ctx, "edittplt_abc123", retab.EditTemplateUpdateRequest{
		Name: &name,
	})
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println((*template)["name"])
}
```

```rust Rust
use retab::models::UpdateEditTemplateRequest;
use retab::resources::edit_templates::UpdateParams;
use retab::Retab;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Retab::new(std::env::var("RETAB_API_KEY")?);

    let template = client
        .edit_templates()
        .update(
            "edittplt_abc123",
            UpdateParams {
                body: UpdateEditTemplateRequest {
                    name: Some("W-9 Form (2024)".into()),
                    form_fields: None,
                },
            },
        )
        .await?;
    println!("{}", template.name);
    Ok(())
}
```
