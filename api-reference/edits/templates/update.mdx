---
title: 'Update Template'
openapi: 'PATCH /v1/edits/templates/{template_id}'
---

Update the name and/or the form fields of an existing template. Both fields are optional; only supplied values are overwritten. Supplying `form_fields` replaces the entire list.

<RequestExample>

```python python
from retab import Retab

client = Retab()

template = client.edits.templates.update(
    "edittplt_abc123",
    name="W-9 Form (2024)",
)
print(template.name)
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();
const template = await client.edits.templates.update("edittplt_abc123", {
    name: "W-9 Form (2024)",
});
console.log(template.name);
```

```bash curl
curl -X PATCH \
  'https://api.retab.com/v1/edits/templates/edittplt_abc123' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{"name": "W-9 Form (2024)"}'
```

</RequestExample>

## Path Parameters

<ParamField path="template_id" type="string" required>
  The unique identifier of the template to update.
</ParamField>

## Request Body

<ParamField body="name" type="string">
  New name for the template.
</ParamField>

<ParamField body="form_fields" type="array[FormField]">
  Replacement list of form fields. When provided, fully replaces the existing list.
</ParamField>
