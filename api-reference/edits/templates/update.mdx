---
title: "Update Template"
openapi: "PATCH /v1/edits/templates/{template_id}"
---

Update the name and/or the form fields of an existing template. Both fields are optional; only supplied values are overwritten. Supplying `form_fields` replaces the entire list.

<RequestExample>

```python python
from retab import Retab

client = Retab()

template = client.edits.templates.update(
    "edittplt_abc123",
    name="W-9 Form (2024)",
)
print(template.name)
```

```javascript javascript
import { Retab } from "@retab/node";

const client = new Retab();
const template = await client.edits.templates.update("edittplt_abc123", {
  name: "W-9 Form (2024)",
});
console.log(template.name);
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	name := "W-9 Form (2024)"
	template, err := client.Edits.Templates.Update(ctx, "edittplt_abc123", retab.EditTemplateUpdateRequest{
		Name: &name,
	})
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println((*template)["name"])
}
```

```bash curl
curl -X PATCH \
  'https://api.retab.com/v1/edits/templates/edittplt_abc123' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{"name": "W-9 Form (2024)"}'
```
