---
title: 'Delete Template'
openapi: 'DELETE /v1/edits/templates/{template_id}'
---

Delete an `EditTemplate` by ID. Existing `Edit` records that referenced this template are preserved but new calls to `POST /v1/edits/templates/fill` with this id will return `404`.

<RequestExample>

```python python
from retab import Retab

client = Retab()
client.edits.templates.delete("edittplt_abc123")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();
await client.edits.templates.delete("edittplt_abc123");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/edits/templates/edittplt_abc123' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```text 204
No Content
```

```json 404
{
  "detail": "Template edittplt_abc123 not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="template_id" type="string" required>
  The unique identifier of the template to delete.
</ParamField>
