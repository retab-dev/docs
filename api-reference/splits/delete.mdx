---
title: 'Delete Split'
openapi: 'DELETE /v1/splits/{split_id}'
---

Delete a persisted `Split` record by ID. Returns `204 No Content` on success.

<RequestExample>

```python python
from retab import Retab

client = Retab()
client.splits.delete("split_01G34H8J2K")
```

```javascript javascript
import { Retab } from '@retab/node';

const client = new Retab();
await client.splits.delete("split_01G34H8J2K");
```

```bash curl
curl -X DELETE \
  'https://api.retab.com/v1/splits/split_01G34H8J2K' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```text 204
No Content
```

```json 404
{
  "detail": "Split not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="split_id" type="string" required>
  The unique identifier of the split to delete.
</ParamField>
