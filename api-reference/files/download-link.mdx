---
title: 'Get Download Link'
openapi: 'GET /v1/files/{file_id}/download-link'
---

Get a temporary signed URL to download the original file. The link expires after 60 minutes.

<RequestExample>

```python python
from retab import Retab

client = Retab()

link = client.files.get_download_link("file_a1b2c3d4e5f6")

print(f"URL: {link.download_url}")
print(f"Expires in: {link.expires_in}")
print(f"Filename: {link.filename}")
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const link = await client.files.getDownloadLink("file_a1b2c3d4e5f6");

console.log(`URL: ${link.downloadUrl}`);
console.log(`Expires in: ${link.expiresIn}`);
console.log(`Filename: ${link.filename}`);
```

```bash curl
curl -X GET \
  'https://api.retab.com/v1/files/file_a1b2c3d4e5f6/download-link' \
  -H "Api-Key: $RETAB_API_KEY"
```

</RequestExample>

<ResponseExample>

```json 200
{
  "download_url": "https://storage.googleapis.com/retab-files/...",
  "expires_in": "60 minutes",
  "filename": "invoice.pdf"
}
```

```json 404
{
  "detail": "File not found"
}
```

</ResponseExample>

## Path Parameters

<ParamField path="file_id" type="string" required>
  The ID of the file to generate a download link for.
</ParamField>

## Response Fields

<ResponseField name="download_url" type="string">
  A signed URL to download the file. Valid for 60 minutes.
</ResponseField>

<ResponseField name="expires_in" type="string">
  Human-readable expiration time (e.g., `"60 minutes"`).
</ResponseField>

<ResponseField name="filename" type="string">
  The filename of the file.
</ResponseField>
