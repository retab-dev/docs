---
title: 'Upload File'
openapi: 'POST /v1/files/upload'
---

Create a direct-to-storage upload session. Upload the file bytes to the returned signed `uploadUrl`, then call `POST /v1/files/upload/{file_id}/complete`. The completed file is returned as `MIMEData` and can be passed directly to extractions, workflows, and other operations.

<RequestExample>

```python python
from retab import Retab
from pathlib import Path

client = Retab()

# Upload from a file path
mime_data = client.files.upload(Path("invoice.pdf"))
print(f"Filename: {mime_data.filename}")
print(f"URL: {mime_data.url}")
```

```javascript javascript
import Retab from 'retab';

const client = new Retab();

const mimeData = await client.files.upload("invoice.pdf");

console.log(`Filename: ${mimeData.filename}`);
console.log(`URL: ${mimeData.url}`);
```

```bash curl
SESSION=$(curl -s -X POST \
  'https://api.retab.com/v1/files/upload' \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{
    "filename": "invoice.pdf",
    "content_type": "application/pdf",
    "size_bytes": 12345
  }')

UPLOAD_URL=$(echo "$SESSION" | jq -r '.uploadUrl')
FILE_ID=$(echo "$SESSION" | jq -r '.fileId')

curl -X PUT "$UPLOAD_URL" \
  -H 'Content-Type: application/pdf' \
  --data-binary '@invoice.pdf'

curl -X POST \
  "https://api.retab.com/v1/files/upload/$FILE_ID/complete" \
  -H "Api-Key: $RETAB_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{}'
```

</RequestExample>

## Request Parameters

<ParamField body="filename" type="string" required>
  The filename including extension (e.g., `"invoice.pdf"`).
</ParamField>

<ParamField body="content_type" type="string">
  MIME type for the object upload. If omitted, Retab guesses from `filename`.
</ParamField>

<ParamField body="size_bytes" type="integer" required>
  Expected file size in bytes. The complete step verifies that storage received this exact size.
</ParamField>

<ParamField body="sha256" type="string">
  Optional SHA-256 checksum. If provided, the complete step must include the same checksum.
</ParamField>

## Response Fields

<ResponseField name="fileId" type="string">
  Unique identifier for the uploaded file, prefixed with `file_`.
</ResponseField>

<ResponseField name="uploadUrl" type="string">
  Short-lived signed URL for uploading file bytes.
</ResponseField>

<ResponseField name="uploadMethod" type="string">
  HTTP method to use for the signed upload. Currently `"PUT"`.
</ResponseField>

<ResponseField name="uploadHeaders" type="object">
  Headers that must be sent with the signed upload.
</ResponseField>

<ResponseField name="mimeData" type="object">
  Durable `MIMEData` reference for the uploaded file. Its `url` is an opaque Retab storage URL and is only usable through Retab APIs.
</ResponseField>

<ResponseField name="expiresAt" type="string">
  Expiration timestamp for the signed upload URL.
</ResponseField>

## Use the uploaded file in later requests

The SDK returns a durable `MIMEData` reference. You can pass that object directly, or pass its Retab storage URL.

<RequestExample>

```python python
mime_ref = client.files.upload("large_document.pdf")

classification = client.classifications.create(
    document=mime_ref.url,
    model="retab-micro",
    categories=[
        {"name": "invoice", "description": "Invoices with billing details and totals"},
        {"name": "other", "description": "Any other document"},
    ],
)

# This also works:
classification = client.classifications.create(
    document=mime_ref,
    model="retab-micro",
    categories=[
        {"name": "invoice", "description": "Invoices with billing details and totals"},
        {"name": "other", "description": "Any other document"},
    ],
)
```

```javascript javascript
const mimeRef = await client.files.upload("large_document.pdf");

const classification = await client.classifications.create({
  document: mimeRef.url,
  model: "retab-micro",
  categories: [
    { name: "invoice", description: "Invoices with billing details and totals" },
    { name: "other", description: "Any other document" },
  ],
});

// This also works:
const sameClassification = await client.classifications.create({
  document: mimeRef,
  model: "retab-micro",
  categories: [
    { name: "invoice", description: "Invoices with billing details and totals" },
    { name: "other", description: "Any other document" },
  ],
});
```

</RequestExample>

`https://storage.retab.com/file_...` URLs are opaque Retab references, not public download links. Retab resolves them against the authenticated caller's organization before processing.
