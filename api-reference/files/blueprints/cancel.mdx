---
title: "Cancel File Blueprint"
openapi: "POST /v1/files/blueprints/{blueprint_id}/cancel"
---

Cancel an in-flight background file blueprint.

<RequestExample>

```python Python
from retab import Retab

client = Retab()

blueprint = client.files.create_blueprint_cancel("fbp_a1b2c3d4e5f6")

print(blueprint.status)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const blueprint = await client.files.create_blueprint_cancel("fbp_a1b2c3d4e5f6");

console.log(blueprint.status);
```

```go Go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	blueprint, err := client.Files.CreateBlueprintCancel(context.Background(), "fbp_a1b2c3d4e5f6")
	if err != nil {
		log.Fatal(err)
	}

	fmt.Println(*blueprint.Status)
}
```
