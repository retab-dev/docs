---
title: 'List Files'
openapi: 'GET /v1/files'
---

List uploaded files with pagination and optional filtering by filename or MIME type.

<RequestExample>

```python python
from retab import Retab

client = Retab()

# List recent files
files = client.files.list(limit=20)
for f in files:
    print(f"{f.id}: {f.filename}")

# Filter by filename prefix and MIME type
files = client.files.list(
    filename="invoice",
    mime_type="application/pdf",
    limit=50
)
```

```go go
package main

import (
	"context"
	"fmt"
	"log"

	retab "github.com/retab-dev/retab/clients/go"
)

func main() {
	ctx := context.Background()

	client, err := retab.NewClient("")
	if err != nil {
		log.Fatal(err)
	}

	// List recent files
	files, err := client.Files.List(ctx, &retab.ListFilesParams{
		ListParams: retab.ListParams{Limit: 20},
	})
	if err != nil {
		log.Fatal(err)
	}
	for _, f := range files.Data {
		fmt.Printf("%s: %s\n", f["id"], f["filename"])
	}

	// Filter by filename prefix and MIME type
	filtered, err := client.Files.List(ctx, &retab.ListFilesParams{
		ListParams: retab.ListParams{Filename: "invoice", Limit: 50},
		MIMEType:   "application/pdf",
	})
	if err != nil {
		log.Fatal(err)
	}
	_ = filtered
}
```

## Query Parameters

<ParamField query="before" type="string">
  Cursor for backward pagination.
</ParamField>

<ParamField query="after" type="string">
  Cursor for forward pagination.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Maximum number of files to return. Range: 1-100.
</ParamField>

<ParamField query="order" type="string" default="desc">
  Sort order. Valid values: `asc`, `desc`.
</ParamField>

<ParamField query="filename" type="string">
  Filter by filename prefix (case-insensitive).
</ParamField>

<ParamField query="mime_type" type="string">
  Filter by MIME type (e.g., `application/pdf`, `image/png`).
</ParamField>

<ParamField query="sort_by" type="string" default="created_at">
  Field to sort results by.
</ParamField>

## Response Fields

Each item in the response array is a File object:

<ResponseField name="id" type="string">
  Unique file identifier, prefixed with `file_`.
</ResponseField>

<ResponseField name="object" type="string">
  Always `"file"`.
</ResponseField>

<ResponseField name="filename" type="string">
  The original filename.
</ResponseField>

<ResponseField name="page_count" type="integer | null">
  Number of pages (if applicable).
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 creation timestamp.
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 last update timestamp.
</ResponseField>
