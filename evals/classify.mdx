---
title: "Classify Evals"
description: "How to use the processing-only SDK surface under client.evals.classify."
---

`client.evals.classify` now exposes a single processing entry point: `process(...)`.

<CodeGroup>

```python Python
from retab import Retab
from retab.types.mime import MIMEData

client = Retab()

result = client.evals.classify.process(
    eval_id="classify_eval_123",
    document=MIMEData(
        filename="document.pdf",
        url="data:application/pdf;base64,JVBERi0xLjQKJ...",
    ),
)

print(result.classification)
```

```typescript TypeScript
import { Retab } from "@retab/node";

const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

const result = await client.evals.classify.process({
  eval_id: "classify_eval_123",
  document: {
    filename: "document.pdf",
    url: "data:application/pdf;base64,JVBERi0xLjQKJ...",
  },
});

console.log(result.classification);
```
