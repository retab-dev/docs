## title: Quickstart

---

While Retab is built around evals, the most basic workflow is extracting structured data from a document. The easiest way to access the API is through the Python or Node SDK.


<Tip>
  Find your API Key in the [dashboard settings](https://retab.com/dashboard/settings).
</Tip>

## Using the Retab SDK

<Steps>
  <Step title="Install the SDK" stepNumber={1}>
    <CodeGroup>
      ```python Python
      pip install retab
      ```

      ```javascript JavaScript
      npm install @retab/node
      ```
    </CodeGroup>
  </Step>

  <Step title="Generate a Schema" stepNumber={2}>
    <CodeGroup>
      ```python Python
      from retab import Retab

      client = Retab()

      schema_response = client.schemas.generate(
          documents=["Invoice.pdf"],
          model="retab-small",
      )
      ```

      ```javascript JavaScript
      import { Retab } from '@retab/node';

      const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

      const schema_response = await client.schemas.generate({
          documents: ["Invoice.pdf"],
          model: "retab-small",        // or any model your plan supports
      });
      ```

      ```curl cURL
      curl https://api.retab.com/v1/schemas/generate \
        -H "Api-Key: $RETAB_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "documents": [
            {
              "filename": "Invoice.pdf",
              "url": "data:application/pdf;base64,JVBERi0xLjQKJfbk/N8KMS..."
            }
          ],
          "model": "retab-small"
        }'

      ```
    </CodeGroup>
  </Step>

  <Step title="Extract Data" stepNumber={3}>
    <CodeGroup>
      ```python Python
      from retab import Retab

      client = Retab()

      extraction_response = client.documents.extract(
          json_schema = schema_response.json_schema,
          document = "Invoice.pdf",
          model="retab-micro",
      )

      print(extraction_response.data)
      print(extraction_response.likelihoods)
      print(extraction_response.extraction_id)
      ```

      ```javascript JavaScript
      import { Retab } from '@retab/node';

      const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

      const extraction_response = await client.documents.extract({
          json_schema: schema_response.json_schema,
          document: "Invoice.pdf",
          model: "retab-micro",
      });

      console.log(extraction_response.choices[0].message.parsed);

      console.log(extraction_response.extraction_id);
      ```
