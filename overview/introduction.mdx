Retab is a document processing [worfklow](../workflows/Workflows) builder for the age of AI.

---

Retab transforms any document into structured data, with six core functionalities:

<a href="../api-reference/parses/create" target="_self" className="endpoint-link">parse -></a>\
Convert any file (PDFs, Excel, emails, images) into LLM-ready markdown.

<a href="../api-reference/documents/extract" target="_self" className="endpoint-link">extract -></a>\
Extract structured JSON from documents using your defined schema.

<a href="../api-reference/edit/agent/fill" target="_self" className="endpoint-link">edit -></a>\
Modify document content while preserving formatting.

<a href="../api-reference/documents/split" target="_self" className="endpoint-link">split -></a>\
Intelligently split documents into logical sections.

<a href="../api-reference/partitions/create" target="_self" className="endpoint-link">partition -></a>\
Partition repeated records in a document into chunks using a unique identifier

<a href="../api-reference/classifications/create" target="_self" className="endpoint-link">classify -></a>\
Categorize documents based on content and type.

<br />

## Quickstart

The most basic workflow is extracting structured data from a document. The easiest way to access the API is through the Python or Node SDK.

<Tip>
  Find your API key in the [dashboard settings](https://retab.com/dashboard/settings).
</Tip>

<Steps>
  <Step title="Install the SDK" stepNumber={1}>
    <CodeGroup>
      ```python Python
      pip install retab
      ```

      ```javascript JavaScript
      npm install @retab/node
      ```
    </CodeGroup>
  </Step>

  <Step title="Generate a Schema" stepNumber={2}>
    <CodeGroup>
      ```python Python
      from retab import Retab

      client = Retab()

      schema_response = client.schemas.generate(
          documents=["Invoice.pdf"],
          model="retab-small",
      )
      ```

      ```javascript JavaScript
      import { Retab } from '@retab/node';

      const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

      const schemaResponse = await client.schemas.generate({
          documents: ["Invoice.pdf"],
          model: "retab-small",
      });
      ```

      ```curl cURL
      curl https://api.retab.com/v1/schemas/generate \
        -H "Api-Key: $RETAB_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "documents": [
            {
              "filename": "Invoice.pdf",
              "url": "data:application/pdf;base64,JVBERi0xLjQKJfbk/N8KMS..."
            }
          ],
          "model": "retab-small"
        }'
      ```
    </CodeGroup>
  </Step>

  <Step title="Extract Data" stepNumber={3}>
    <CodeGroup>
      ```python Python
      from retab import Retab

      client = Retab()

      extraction = client.extractions.create(
          json_schema=schema_response.json_schema,
          document="Invoice.pdf",
          model="retab-micro",
      )

      print(extraction.id)
      print(extraction.output)
      print(extraction.consensus.likelihoods)
      ```

      ```javascript JavaScript
      import { Retab } from '@retab/node';

      const client = new Retab({ apiKey: process.env.RETAB_API_KEY });

      const extraction = await client.extractions.create({
          json_schema: schemaResponse.json_schema,
          document: "Invoice.pdf",
          model: "retab-micro",
      });

      console.log(extraction.id);
      console.log(extraction.output);
      console.log(extraction.consensus.likelihoods);
      ```

      ```curl cURL
      curl -X POST https://api.retab.com/v1/extractions \
        -H "Api-Key: $RETAB_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "document": {
            "filename": "Invoice.pdf",
            "url": "data:application/pdf;base64,JVBERi0xLjQKJfbk/N8KMS..."
          },
          "model": "retab-micro",
          "json_schema": {
            "title": "InvoiceSchema",
            "type": "object",
            "properties": {
              "invoice_number": { "type": "string" },
              "date_due": { "type": "string" },
              "total": { "type": "number" }
            },
            "required": ["invoice_number", "date_due", "total"]
          },
          "n_consensus": 1,
          "metadata": {
            "source": "introduction_quickstart"
          }
        }'
      ```
    </CodeGroup>
  </Step>
</Steps>
